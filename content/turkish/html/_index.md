## 🚀 Hızlı Başlangıç

### Kurulum
```bash
# NuGet Paket Yöneticisi aracılığıyla yükleyin
Install-Package Aspose.Html
```

### Temel HTML'den PDF'e Dönüştürme
```csharp
using Aspose.Html;
using Aspose.Html.Converters;
using Aspose.Html.Rendering.Pdf;

// HTML belgesini yükle
var document = new HTMLDocument("input.html");

// PDF'ye dönüştür
Converter.ConvertHTML(document, new PdfRenderingOptions(), "output.pdf");
```

### Temel HTML'den PNG'ye Dönüştürme
```csharp
using Aspose.Html;
using Aspose.Html.Converters;
using Aspose.Html.Rendering.Image;

// HTML belgesini yükle
var document = new HTMLDocument("input.html");

// PNG'ye dönüştür
Converter.ConvertHTML(document, new ImageRenderingOptions(), "output.png");
```
