## مقدمة

إنشاء صور مصغرة لكل صفحة في ملف PDF طريقة رائعة لتحسين تصفح المستندات ومعاينتها. سواء كنت تُطوّر نظام إدارة مستندات أو تُنظّم ملفات PDF ببساطة، فإن إنشاء الصور المصغرة يُوفّر عليك الوقت ويُحسّن تجربة المستخدم. في هذا الدليل، سنستكشف كيفية استخدام Aspose.PDF لـ .NET لإنشاء صور مصغرة تلقائيًا لكل صفحة من ملفات PDF.

## المتطلبات الأساسية

قبل أن نتعمق في الكود، تأكد من أن لديك ما يلي:

1. المعرفة الأساسية بلغة C# أو .NET: ستساعدك المعرفة بلغة C# على فهم الكود بشكل أفضل.
2. Visual Studio: قم بتثبيت IDE هذا لكتابة وتشغيل التعليمات البرمجية الخاصة بك.
3. Aspose.PDF لمكتبة .NET: قم بتنزيل المكتبة وتثبيتها من [توثيق Aspose.PDF](https://reference.aspose.com/pdf/net/).
4. ملفات PDF: قم بإعداد بعض ملفات PDF في دليل عمل مخصص للاختبار.

## البدء: استيراد الحزم الضرورية

للاستفادة من وظائف Aspose.PDF، ابدأ بتضمين المساحات المطلوبة في أعلى ملف C# الخاص بك:

```csharp
using Aspose.Pdf.Devices;
using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using System.Text;
```

توفر هذه المساحات الأسماء إمكانية الوصول إلى الفئات والطرق اللازمة لعملياتنا.

## الخطوة 1: إعداد دليل المستندات الخاص بك

أولاً، حدد المسار إلى دليل المستندات الذي يتم تخزين كافة ملفات PDF فيه:

```csharp
string dataDir = "YOUR_DOCUMENT_DIRECTORY"; // استبدله بمسار الدليل الفعلي الخاص بك
```

تأكد من الاستبدال `"YOUR_DOCUMENT_DIRECTORY"` مع المسار الفعلي لملفات PDF الخاصة بك، حيث أن هذه الخطوة ضرورية لتحديد موقع الملفات.

## الخطوة 2: استرداد أسماء ملفات PDF

بعد ذلك، استرجاع أسماء جميع ملفات PDF في مجلدك. سيسمح لنا هذا بمراجعة كل ملف لاحقًا:

```csharp
string[] fileEntries = Directory.GetFiles(dataDir, "*.pdf");
```

استخدام `Directory.GetFiles`نقوم بتصفية والحصول على ملفات PDF فقط، مما يضمن جمع كل المستندات ذات الصلة.

## الخطوة 3: التكرار خلال كل ملف PDF

الآن، سوف ننتقل عبر كل ملف ونفتحه لإنشاء صور مصغرة لصفحاته:

```csharp
foreach (string filePath in fileEntries)
{
    Document pdfDocument = new Document(filePath);
    // سيتم إجراء المعالجة الإضافية هنا
}
```

في هذه الحلقة، نفتح كل ملف PDF باستخدام `Document` الصف يستعد لمعالجة صفحاته.

## الخطوة 4: إنشاء صور مصغرة لكل صفحة

سننشئ صورة مصغّرة لكل صفحة في ملف PDF. لنشرح ذلك خطوة بخطوة.

### الخطوة 4.1: تهيئة تدفق الملفات لكل صورة مصغرة

في حلقتنا، قم بإعداد مجرى لحفظ كل صورة مصغرة:

```csharp
for (int pageCount = 1; pageCount <= pdfDocument.Pages.Count; pageCount++)
{
    using (FileStream imageStream = new FileStream(Path.Combine(dataDir, $"Thumbnails_{Path.GetFileNameWithoutExtension(filePath)}_{pageCount}.jpg"), FileMode.Create))
    {
        // سيتم إجراء المعالجة الإضافية هنا
    }
}
```

يؤدي هذا إلى إنشاء ملف JPG جديد لكل صورة مصغرة، وتسميته بشكل فريد استنادًا إلى اسم ملف PDF الأصلي ورقم الصفحة.

### الخطوة 4.2: تحديد الدقة

بعد ذلك، حدد دقة الصور المصغرة. الدقة العالية تُنتج صورًا أوضح، لكنها تزيد من حجم الملف.

```csharp
Resolution resolution = new Resolution(300);
```

تعتبر الدقة 300 نقطة في البوصة معيارًا للصور عالية الجودة، ولكن لا تتردد في تعديلها حسب الحاجة.

### الخطوة 4.3: إعداد JpegDevice

الآن، قم بإعداد `JpegDevice`، الذي سيقوم بتحويل صفحات PDF إلى صور:

```csharp
using (JpegDevice jpegDevice = new JpegDevice(45, 59, resolution, 100))
{
    // سيتم إجراء المعالجة الإضافية هنا
}
```

هنا، نحدد أبعاد الصور المصغرة (45×59 بكسل) وجودتها. عدّل هذه القيم وفقًا لاحتياجات تطبيقك.

### الخطوة 4.4: معالجة كل صفحة

بمجرد وضع كل شيء في مكانه، قم بمعالجة كل صفحة من ملف PDF وحفظ الصورة المصغرة الناتجة:

```csharp
jpegDevice.Process(pdfDocument.Pages[pageCount], imageStream);
```

يقوم هذا الخط بتحويل صفحة PDF المحددة إلى تنسيق JPEG ويكتبها مباشرة إلى `imageStream`.

### الخطوة 4.5: إغلاق البث

أخيرًا، بعد معالجة كل صفحة، أغلق البث لتحرير الموارد:

```csharp
imageStream.Close();
```

يعد إغلاق الدفق أمرًا ضروريًا لمنع تسرب الذاكرة والتأكد من حفظ كافة التغييرات.

## خاتمة

يُحسّن إنشاء الصور المصغرة لملفات PDF تفاعل المستخدم مع المستندات بشكل ملحوظ. باستخدام Aspose.PDF لـ .NET، تُصبح هذه العملية سهلة وفعّالة. باتباع هذا الدليل، يُمكنك بسهولة دمج الصور المصغرة لملفات PDF في مشاريعك، مما يُسهّل التنقل ويُحسّن إمكانية الوصول.

## الأسئلة الشائعة

### ما هو Aspose.PDF؟  
Aspose.PDF هي مكتبة قوية لإنشاء وتحرير وتحويل مستندات PDF في تطبيقات .NET.

### هل Aspose.PDF مجاني؟  
Aspose.PDF هو منتج تجاري، ولكن يمكنك تنزيل نسخة تجريبية مجانية من موقعهم [موقع إلكتروني](https://releases.aspose.com/).

### هل يمكنني تخصيص أبعاد الصورة المصغرة؟  
نعم، يمكنك تعديل معلمات العرض والارتفاع في `JpegDevice` منشئ لتعيين أحجام الصور المصغرة المطلوبة.

### هل هناك اعتبارات تتعلق بالأداء عند تحويل ملفات PDF كبيرة الحجم؟  
نعم، قد تستغرق معالجة الملفات الكبيرة وقتًا أطول، وذلك حسب الدقة وعدد الصفحات. تحسين هذه المعلمات يُحسّن الأداء.

### أين يمكنني العثور على المزيد من الموارد والدعم؟
