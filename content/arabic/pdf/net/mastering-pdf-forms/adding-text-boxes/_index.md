## مقدمة

في ظلّ العصر الرقميّ الحالي، يُعدّ تحسين تجربة المستخدم من خلال المستندات التفاعلية أمرًا بالغ الأهمية. فنماذج PDF التفاعلية لا تُسهّل جمع البيانات فحسب، بل تُشرك المستخدمين أيضًا بطريقة لا تُمكنها المستندات الثابتة. تُعدّ Aspose.PDF for .NET مكتبة فعّالة مُصمّمة لمساعدة المُطوّرين على دمج حقول النماذج المُختلفة في مستندات PDF بسهولة. ومن بين هذه الحقول، تُعدّ مربعات النصّ مفيدةً بشكلٍ خاصّ لجمع مُدخلات المستخدم بطريقة مُنظّمة. في هذا البرنامج التعليمي، سنشرح بالتفصيل عملية إضافة مربع نصّ إلى ملف PDF باستخدام Aspose.PDF for .NET، مع ضمان فهمك الشامل لكلّ خطوة.

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك ما يلي:

1. المعرفة الأساسية بلغة C#: ستساعدك المعرفة بقواعد لغة C# وبنيتها على متابعة الكود.
2. تم تثبيت Aspose.PDF لـ .NET: قم بتنزيل مكتبة Aspose.PDF وتثبيتها من [موقع](https://releases.aspose.com/pdf/net/).
3. بيئة التطوير: استخدم بيئة التطوير المتكاملة مثل Visual Studio للترميز والاختبار.
4. .NET Framework: تأكد من تثبيت إصدار متوافق من .NET Framework.

مع توفر هذه المتطلبات الأساسية، نحن جاهزون للبدء في الترميز!

### افتح IDE الخاص بك

قم بتشغيل بيئة التطوير المفضلة لديك (يوصى باستخدام Visual Studio).

### إنشاء مشروع جديد

قم بإعداد مشروع C# جديد عن طريق تحديد "إنشاء مشروع جديد" واختيار قالب تطبيق وحدة التحكم للبساطة.

### تثبيت حزمة Aspose.PDF

دمج مكتبة Aspose.PDF في مشروعك باستخدام مدير الحزم NuGet. في وحدة تحكم مدير الحزم، شغّل:

```bash
Install-Package Aspose.PDF
```

## استيراد مساحة اسم Aspose.PDF

في أعلى ملف البرنامج الرئيسي الخاص بك (عادةً `Program.cs`), تتضمن مساحات الأسماء التالية:

```csharp
using System.IO;
using System;
using Aspose.Pdf.Forms;
using Aspose.Pdf;
using Aspose.Pdf.Annotations;
```

هذا الإعداد يجهزك للمهام المثيرة القادمة!

الآن بعد أن أصبح كل شيء جاهزًا، دعنا نستعرض الخطوات اللازمة لإضافة مربع نص إلى مستند PDF الخاص بك.

## الخطوة 1: تحديد دليل المستندات الخاص بك

أولاً، حدد الدليل الذي يوجد فيه مستند PDF الخاص بك. استبدل `"YOUR DOCUMENT DIRECTORY"` مع المسار الفعلي:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## الخطوة 2: افتح مستند PDF 

قم بتحميل ملف PDF إلى مثيل `Document` فصل:

```csharp
Document pdfDocument = new Document(dataDir + "TextField.pdf");
```

تأكد من ذلك `"TextField.pdf"` موجود في الدليل المحدد.

## الخطوة 3: إنشاء حقل مربع النص

الآن، دعونا نقوم بإنشاء حقل مربع النص:

```csharp
TextBoxField textBoxField = new TextBoxField(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(100, 200, 300, 300));
```

- جديد `TextBoxField` تم تهيئة الكائن للصفحة الثانية من ملف PDF الخاص بك.
- ال `Rectangle` تحدد المعلمة موضع وحجم مربع النص باستخدام الإحداثيات (x1، y1، x2، y2).

## الخطوة 4: تعيين خصائص حقل مربع النص 

قم بتخصيص مربع النص الخاص بك باستخدام الخصائص التالية:

```csharp
textBoxField.PartialName = "textbox1";
textBoxField.Value = "Text Box";
```

- `PartialName` يعطي معرفًا فريدًا لمربع النص.
- `Value` تعيين النص الافتراضي الذي يظهر داخل المربع.

## الخطوة 5: تخصيص الحدود

دعونا نعمل على تحسين مظهر مربع النص الخاص بنا عن طريق تخصيص حدوده:

```csharp
Border border = new Border(textBoxField);
border.Width = 5; 
border.Dash = new Dash(1, 1);
textBoxField.Border = border;
textBoxField.Color = Aspose.Pdf.Color.FromRgb(System.Drawing.Color.Green);
```

- إنشاء حدود وتحديد عرضها.
- يطبق نمطًا متقطعًا على الحدود.
- تعيين اللون الأخضر لمربع النص.

## الخطوة 6: إضافة مربع النص إلى المستند

الآن، سنضيف حقل مربع النص إلى مستند PDF الخاص بنا:

```csharp
pdfDocument.Form.Add(textBoxField, 1);
```

يقوم هذا السطر بدمج مربع النص في الصفحة الأولى من ملف PDF.

## الخطوة 7: حفظ ملف PDF المعدل

وأخيرًا، احفظ التغييرات بالكود التالي:

```csharp
dataDir = dataDir + "TextBox_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nTextbox field added successfully.\nFile saved at " + dataDir);
```

يحفظ هذا المقطع ملف PDF المُعدَّل باسم جديد. تحقق من مسار الإخراج لملف PDF الذي أنشأته حديثًا!
