## مقدمة

يمكن لملفات PDF التفاعلية أن تُحسّن تفاعل المستخدم بشكل ملحوظ، خاصةً عند استخدام النماذج. يُعد زر الاختيار أحد أكثر العناصر التفاعلية فعالية، إذ يتيح للمستخدمين اختيار خيار واحد من مجموعة خيارات. في هذا البرنامج التعليمي، سنشرح خطوات إنشاء أزرار الاختيار في مستند PDF باستخدام Aspose.PDF لـ .NET. سواءً كنت مطورًا محترفًا أو مبتدئًا، سيساعدك هذا الدليل على فهم كل جزء من الكود.

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك ما يلي:

1. Visual Studio: بيئة التطوير الخاصة بك.
2. Aspose.PDF لـ .NET: قم بتنزيل المكتبة من [موقع Aspose](https://releases.aspose.com/pdf/net/).
3. المعرفة الأساسية بلغة C#: ستساعدك المعرفة بلغة C# على التنقل عبر مقتطفات التعليمات البرمجية.

## إنشاء مشروع جديد

1. افتح Visual Studio.
2. إنشاء مشروع تطبيق وحدة التحكم الجديد.

## إضافة مرجع Aspose.PDF

1. انقر بزر الماوس الأيمن فوق مشروعك في مستكشف الحلول.
2. حدد إدارة حزم NuGet.
3. ابحث عن Aspose.PDF وقم بتثبيت الإصدار الأحدث.

الآن بعد أن قمت بإعداد بيئتك، دعنا ننتقل إلى الكود.

## الخطوة 1: تحديد دليل المستندات الخاص بك

حدد الدليل الذي سيتم حفظ ملف PDF الخاص بك فيه:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // استبدل بالمسار الفعلي الخاص بك
```

## الخطوة 2: إنشاء كائن المستند

إنشاء مثيل لـ `Document` فصل:

```csharp
Document pdfDocument = new Document();
```

## الخطوة 3: إضافة صفحة إلى ملف PDF

أضف صفحة جديدة إلى مستند PDF الخاص بك:

```csharp
pdfDocument.Pages.Add();
```

## الخطوة 4: إنشاء حقل زر الاختيار

إنشاء مثيل `RadioButtonField` كائن للصفحة الأولى:

```csharp
RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);
```

## الخطوة 5: إضافة خيارات إلى زر الاختيار

قم بتحديد الخيارات الخاصة بأزرار الراديو الخاصة بك:

```csharp
radio.AddOption("Option 1", new Rectangle(0, 0, 20, 20));
radio.AddOption("Option 2", new Rectangle(0, 30, 20, 20));
```

يضيف هذا المثال خيارين: "الخيار 1" و"الخيار 2". `Rectangle` يحدد الكائن موضع وحجم كل خيار.

## الخطوة 6: إضافة زر الاختيار إلى نموذج المستند

دمج زر الراديو في نموذج PDF:

```csharp
pdfDocument.Form.Add(radio);
```

## الخطوة 7: حفظ مستند PDF

احفظ مستند PDF الخاص بك في الدليل المحدد:

```csharp
dataDir = dataDir + "RadioButton_out.pdf";
pdfDocument.Save(dataDir);
```

## الخطوة 8: التعامل مع الاستثناءات

تنفيذ معالجة الأخطاء للقبض على أي مشاكل:

```csharp
try
{

```
