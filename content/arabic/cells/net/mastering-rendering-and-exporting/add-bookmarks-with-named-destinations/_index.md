## مقدمة

قد يبدو تصفح ملفات PDF الكبيرة أشبه بالبحث عن إبرة في كومة قش، خاصةً عند إنشائها من جداول بيانات Excel واسعة النطاق. تُتيح الإشارات المرجعية في مستندات PDF التنقل بسلاسة بين أقسام الملف ذات الصلة، مما يُحسّن تجربة المستخدم. سيُرشدك هذا الدليل المُفصّل خلال عملية إضافة إشارات مرجعية ذات وجهات مُسمّاة إلى ملف PDF مُنشأ من ملف Excel باستخدام Aspose.Cells لـ .NET.

## المتطلبات الأساسية للعمل مع Aspose.Cells لـ .NET

قبل التعمق في الكود، من الضروري التأكد من إعداد جميع الأدوات اللازمة للتنفيذ الناجح. إليك المتطلبات الأساسية:

1. Visual Studio: بيئة التطوير المتكاملة المُوصى بها لتطوير .NET. تأكد من تثبيته وتكوينه بشكل صحيح على نظامك.
2. Aspose.Cells لـ .NET: المكتبة الأساسية اللازمة لمعالجة ملفات Excel برمجيًا. يمكنك [قم بتحميله هنا](https://releases.aspose.com/cells/net/)إذا كنت جديدًا في Aspose، فيمكنك البدء بـ [نسخة تجريبية مجانية](https://releases.aspose.com/).
3. إطار عمل .NET: تأكد من استخدام إصدار متوافق من إطار عمل .NET. يدعم Aspose.Cells إصدارات متعددة.
4. المعرفة الأساسية بلغة C#: سيكون الفهم الأساسي للغة C# مفيدًا في متابعة الكود.

مع توفر هذه المكونات، ستكون جاهزًا لبدء إنشاء مستندات PDF مع الإشارات المرجعية!

## إعداد مشروعك

بعد أن تصبح بيئة التطوير جاهزة، يمكنك إنشاء مشروع C# جديد في Visual Studio. للعمل مع وظائف Aspose.Cells، ستحتاج إلى استيراد مساحات الأسماء المطلوبة.

## استيراد مساحات الأسماء المطلوبة

في أعلى ملف C# الخاص بك، أضف ما يلي `using` عبارات للتأكد من أن مشروعك يمكنه الوصول إلى Aspose.Cells لـ .NET:

```csharp
using System;
using System.Collections;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using Aspose.Cells.Rendering;
using System.Drawing.Imaging;
```

توفر هذه المساحات الاسمية إمكانية الوصول إلى الفئات الأساسية التي ستساعدك على معالجة بيانات Excel وتحويلها إلى ملف PDF.

## الخطوة 1: إعداد الدلائل لملفات الإدخال والإخراج

الخطوة الأولى هي تحديد مجلدات ملفات الإدخال والإخراج. هذا يضمن تحديد موقع ملف Excel المصدر وملف PDF الناتج بشكل صحيح.

```csharp
string sourceDir = "Your Document Directory";  // المسار إلى ملفات Excel الخاصة بك
string outputDir = "Your Document Directory"; // المسار الذي سيتم حفظ ملف PDF الناتج فيه
```

وهذا يشبه تنظيم مساحة العمل الخاصة بك قبل البدء بمشروع ما.

## الخطوة 2: تحميل مصنف Excel

الخطوة التالية هي تحميل ملف Excel المصدر. يتيح لك Aspose.Cells تحميل ملف Excel بسهولة إلى `Workbook` الكائن، مما يوفر إمكانية الوصول إلى كافة أوراقه وخلاياه ومحتوياته.

```csharp
Workbook wb = new Workbook(sourceDir + "sampleExcelFile.xlsx");
```

يؤدي هذا إلى فتح المصنف وتجهيزه للمعالجة. يمكنك الآن البدء باستخراج البيانات وتنسيقها لملف PDF.

## الخطوة 3: الوصول إلى ورقة العمل

بعد تحميل المصنف، حان وقت الوصول إلى ورقة العمل التي تحتوي على الخلايا المناسبة للإشارات المرجعية. في هذا المثال، سنعمل مع ورقة العمل الأولى:

```csharp
Worksheet ws = wb.Worksheets[0]; // الوصول إلى ورقة العمل الأولى
```

هذه الخطوة تُنشئ لوحةً لإشاراتك المرجعية. كل خليةٍ ستُشير إليها لإشارة مرجعية ستُشتق من ورقة العمل هذه.

## الخطوة 4: إنشاء إشارات مرجعية بوجهات مُسمّاة

في هذه المرحلة، يمكننا البدء بإنشاء الإشارات المرجعية. الإشارات المرجعية هي روابط تُسهّل الوصول السريع إلى أجزاء مُحددة من مستندك. في هذا المثال، سننشئ إشارة مرجعية للخلية "C5".

### إنشاء إشارة مرجعية لخلية واحدة

لإنشاء إشارة مرجعية، يجب عليك أولاً الوصول إلى الخلية التي تريد الارتباط بها. بعد ذلك، ستنشئ `PdfBookmarkEntry` وربطها بموضع الخلية.

```csharp
Cell cell = ws.Cells["C5"];
PdfBookmarkEntry bookmarkEntry = new PdfBookmarkEntry();
bookmarkEntry.Text = "Bookmark for C5"; // نص الإشارة المرجعية
bookmarkEntry.Destination = cell;  // ربط الإشارة المرجعية بالخلية
bookmarkEntry.DestinationName = "AsposeCells--" + cell.Name; // اسم الوجهة الفريد
```

اعتبر هذا بمثابة تحديد نقطة في المستند يمكنك الرجوع إليها بنقرة واحدة. يمكنك تعيين أي نص للإشارة المرجعية (مثل "إشارة مرجعية لـ C5") وربطه بخلية محددة.

### إضافة إشارات مرجعية فرعية لتحسين التنقل

يمكنك تحسين تجربة المستخدم بإضافة إشارات مرجعية فرعية متفرعة من الإشارة المرجعية الرئيسية. يمكن لهذه الإشارات المرجعية الفرعية أن تشير إلى مناطق مختلفة ضمن ورقة العمل نفسها أو إلى أوراق أخرى.

```csharp
cell = ws.Cells["G56"];
PdfBookmarkEntry subbookmarkEntry1 = new PdfBookmarkEntry();
subbookmarkEntry1.Text = "Sub-Bookmark 1"; // نص الإشارة المرجعية الفرعية الأولى
subbookmarkEntry1.Destination = cell;
subbookmarkEntry1.DestinationName = "AsposeCells--" + cell.Name;

cell = ws.Cells["L4"];
PdfBookmarkEntry subbookmarkEntry2 = new PdfBookmarkEntry();
subbookmarkEntry2.Text = "Sub-Bookmark 2"; // نص الإشارة المرجعية الفرعية الثانية
subbookmarkEntry2.Destination = cell;
subbookmarkEntry2.DestinationName = "AsposeCells--" + cell.Name;
```

تعمل هذه الإشارات المرجعية الفرعية كإرشادات إضافية للتنقل عبر المستند، تمامًا مثل الفصول في الكتاب.

### تجميع الإشارات المرجعية الفرعية تحت إشارة مرجعية رئيسية

لإنشاء هيكل هرمي، يمكنك إضافة هذه الإشارات المرجعية الفرعية أسفل الإشارة المرجعية الرئيسية. هذا يُسهّل على المستخدمين التنقل بين الأقسام المختلفة.

```csharp
ArrayList list = new ArrayList();
list.Add(subbookmarkEntry1);
list.Add(subbookmarkEntry2);
bookmarkEntry.SubEntry = list; // إضافة إشارات مرجعية فرعية إلى الإشارة المرجعية الرئيسية
```

يؤدي هذا إلى إنشاء بنية تشبه الشجرة حيث يمكن أن تحتوي كل إشارة مرجعية على إشارات مرجعية فرعية متعددة.

## الخطوة 5: حفظ ملف PDF مع الإشارات المرجعية

### ضبط خيارات حفظ PDF

قبل حفظ المستند بصيغة PDF، يجب تحديد خيارات الحفظ والتأكد من تضمين الإشارات المرجعية. سنستخدم `PdfSaveOptions` لهذا الغرض.

```csharp
PdfSaveOptions opts = new PdfSaveOptions();
opts.Bookmark = bookmarkEntry;  // تعيين الإشارات المرجعية لملف PDF
```

يخبر هذا Aspose.Cells بإنشاء ملف PDF يتضمن الإشارات المرجعية التي أنشأناها للتو.

### حفظ المستند

الآن بعد أن تم تعيين الإشارات المرجعية، يمكننا حفظ المصنف بتنسيق PDF.

```csharp
wb.Save(outputDir + "outputWithBookmarks.pdf", opts);
```

يؤدي هذا إلى إنشاء ملف PDF النهائي مع إشارات مرجعية قابلة للنقر، مما يسمح للمستخدمين بالانتقال بسرعة إلى أقسام معينة من المستند.

## خاتمة

باتباع هذه الخطوات البسيطة، نجحت في إنشاء ملف PDF يحتوي على إشارات مرجعية ووجهات مُسمّاة من ملف Excel باستخدام Aspose.Cells لـ .NET. لا تُحسّن إمكانية إضافة الإشارات المرجعية تجربة المستخدم فحسب، بل تُحسّن أيضًا كفاءة التنقل داخل المستندات الكبيرة. سواء كنت تعمل على تقرير أو دليل أو عرض تقديمي، فإن إضافة الإشارات المرجعية إلى الأقسام الرئيسية سيساعد قرّاءك على تحقيق أقصى استفادة من مستندك.
