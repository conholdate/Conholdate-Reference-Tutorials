## مقدمة

عند تنسيق جداول البيانات، يُعدّ اتجاه الصفحة جانبًا بالغ الأهمية، ولكنه غالبًا ما يُغفل. فطريقة محاذاة المحتوى تؤثر بشكل كبير على سهولة القراءة والشكل العام لمستندك. في هذا الدليل، سنستكشف كيفية ضبط اتجاه الصفحة في ورقة عمل Excel باستخدام Aspose.Cells لـ .NET.

## المتطلبات الأساسية

قبل أن نبدأ، تأكد من أن لديك ما يلي:

1. فيجوال ستوديو: تأكد من تثبيته. إذا لم يكن كذلك، فقم بتنزيله من [صفحة تنزيلات Visual Studio](https://visualstudio.microsoft.com/vs/).
2. Aspose.Cells لـ .NET: قم بتنزيل المكتبة وتثبيتها من [صفحة تنزيل Aspose](https://releases.aspose.com/cells/net/)يمكنك أيضًا البدء بـ [نسخة تجريبية مجانية](https://releases.aspose.com/).
3. المعرفة الأساسية بلغة C#: ستكون المعرفة بلغة C# مفيدة، حيث ستكون أمثلتنا بهذه اللغة.

الآن بعد أن قمنا بإعداد كل شيء، فلنبدأ باستيراد الحزم الضرورية.

## استيراد الحزم

لبدء البرمجة، علينا استيراد مكتبة Aspose.Cells إلى مشروعنا. اتبع الخطوات التالية:

### الخطوة 1: افتح Visual Studio

شغّل Visual Studio وأنشئ مشروع C# جديدًا. يمكنك اختيار تطبيق وحدة التحكم أو تطبيق نماذج Windows حسب تفضيلاتك.

### الخطوة 2: إضافة المراجع

في مستكشف الحلول، انقر بزر الماوس الأيمن على مشروعك، ثم اختر "إدارة حزم NuGet"، وابحث عن مكتبة Aspose.Cells. ثبّتها للوصول إلى جميع وظائفها.

### الخطوة 3: استيراد المكتبة

في ملف البرنامج الرئيسي الخاص بك (عادةً `Program.cs`), قم بتضمين التوجيه التالي في الأعلى:

```csharp
using System.IO;
using Aspose.Cells;
using System;
```

سيمنحك هذا إمكانية الوصول إلى جميع الفئات والطرق التي يوفرها Aspose.Cells.

الآن، دعنا نستعرض عملية تعيين اتجاه الصفحة إلى عمودي في ورقة عمل Excel.

## الخطوة 1: تحديد دليل المستندات

أولاً، حدد المسار لتخزين ملف Excel الخاص بك:

```csharp
string dataDir = "Your Document Directory";
```

يستبدل `"Your Document Directory"` مع مسار فعلي، مثل `"C:\\Documents\\"`، حيث تريد حفظ ملف Excel الناتج.

## الخطوة 2: إنشاء كائن مصنف

بعد ذلك، أنشئ نسخة جديدة من مصنف العمل. سيكون هذا الكائن مساحة عملك للتعامل مع جداول البيانات:

```csharp
Workbook workbook = new Workbook();
```

من خلال إنشاء مثيل `Workbook`لقد قمت بإنشاء ملف Excel جديد في الذاكرة.

## الخطوة 3: الوصول إلى ورقة العمل الأولى

الآن، قم بالوصول إلى ورقة العمل الأولى حيث ستحدد اتجاه الصفحة:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

يسترجع هذا السطر ورقة العمل الأولى في المصنف (لاحظ أن أوراق العمل مفهرسة بالصفر).

## الخطوة 4: ضبط الاتجاه إلى الوضع الرأسي

بعد أن أصبحت ورقة العمل الخاصة بك جاهزة، قم بتعيين اتجاه الصفحة باستخدام سطر التعليمات البرمجية التالي:

```csharp
worksheet.PageSetup.Orientation = PageOrientationType.Portrait;
```

لقد قمت الآن بتعيين ورقة العمل الخاصة بك إلى الاتجاه الرأسي بنجاح، مما يؤدي إلى تنظيم المحتوى الخاص بك بشكل عمودي.

## الخطوة 5: حفظ المصنف

وأخيرًا، احفظ التغييرات التي أجريتها على ملف Excel للتأكد من عدم ضياع عملك:

```csharp
workbook.Save(dataDir + "PageOrientation_out.xls");
```

يؤدي هذا إلى حفظ المصنف تحت الاسم `PageOrientation_out.xls` في الدليل المحدد.

## خاتمة

تهانينا! لقد تعلمت كيفية تطبيق اتجاه الصفحة في ورقة عمل باستخدام Aspose.Cells لـ .NET. إنها عملية بسيطة تُحسّن سهولة قراءة جداول بياناتك واحترافيتها.

Aspose.Cells هي مكتبة مدفوعة، ولكن يمكنك البدء بـ [نسخة تجريبية مجانية](https://releases.aspose.com/) لاستكشاف ميزاته.
