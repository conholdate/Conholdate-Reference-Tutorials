## 使用 Aspose.CAD for .NET 將 CAD 匯出為光柵影像  
想要將 CAD 佈局轉換為令人驚嘆的光柵影像嗎？想像一下：您有一個複雜的 CAD 設計，但您的客戶想要一個簡單的 JPEG 預覽。 Aspose.CAD for .NET 讓這項任務變得輕而易舉。借助其強大的 API，您可以將 DWG 或 DXF 等 CAD 檔案匯出為 JPEG、PNG 和 BMP 等格式。  

方法如下：載入您的 CAD 文件，配置您的光柵化選項，並指定輸出格式。瞧！您的 CAD 傑作現在是清晰的高品質影像，可以隨時分享。最好的部分？它可以快速且輕鬆地實現。

## 在 Aspose.CAD for .NET 中將 DGN 轉換為 PDF  
您是否曾經為將 DGN 檔案轉換為 PDF 而苦惱？可以將其想像為將藍圖轉變為精緻的便攜式文件。使用 Aspose.CAD for .NET，您將免去麻煩並直接獲得結果。該庫支援精確的 DGN 到 PDF 轉換，保留每個細節的完整性 - 非常適合共享專業設計。  

以下是簡要概述：匯入 DGN 文件，設定渲染選項，然後將輸出儲存為 PDF。這種方法確保了準確性，同時減少了手工工作。此外，它足夠靈活，適合從初學者到 CAD 專家的各個級別的開發人員。
