## परिचय

क्या आप अपने Word दस्तावेज़ में डिजिटल हस्ताक्षर जोड़ना चाहते हैं? Aspose.Words for .NET के साथ, यह आपके विचार से कहीं ज़्यादा आसान है! यह ट्यूटोरियल आपको अपना वातावरण सेट अप करने, हस्ताक्षर लाइन जोड़ने और अपने दस्तावेज़ पर डिजिटल रूप से हस्ताक्षर करने के चरणों के बारे में बताएगा। चलिए शुरू करते हैं!

## आवश्यक शर्तें

कोड में आगे बढ़ने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1. .NET के लिए Aspose.Words - [यहाँ पर डाउनलोड करो](https://releases.aspose.com/words/net/).
2. .NET विकास वातावरण - विजुअल स्टूडियो इस कार्य के लिए आदर्श है।
3. हस्ताक्षर करने के लिए दस्तावेज़ - आप एक नया वर्ड दस्तावेज़ बना सकते हैं या किसी मौजूदा दस्तावेज़ का उपयोग कर सकते हैं।
4. प्रमाणपत्र फ़ाइल - A `.pfx` डिजिटल हस्ताक्षर के लिए फ़ाइल आवश्यक है।
5. हस्ताक्षर लाइन छवि (वैकल्पिक) - आप हस्ताक्षर के लिए एक छवि फ़ाइल शामिल कर सकते हैं।

## आवश्यक नामस्थान आयात करें

Aspose.Words की कार्यक्षमताओं का उपयोग करने के लिए, आपको निम्नलिखित नामस्थानों को आयात करना होगा:

```csharp
using System;
using System.IO;
using Aspose.Words;
using Aspose.Words.Drawing;
using Aspose.Words.Saving;
using Aspose.Words.Signing;
```

## चरण 1: दस्तावेज़ निर्देशिका सेट अप करना

अपनी दस्तावेज़ निर्देशिका का पथ निर्धारित करके शुरुआत करें। यहीं पर आप अपने दस्तावेज़ सहेजेंगे और प्राप्त करेंगे।

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // अपना दस्तावेज़ निर्देशिका पथ निर्दिष्ट करें
```

## चरण 2: नया दस्तावेज़ बनाना

अब, एक नया वर्ड डॉक्यूमेंट बनाएँ। यह डॉक्यूमेंट आपकी हस्ताक्षर लाइन के लिए कैनवास का काम करेगा।

```csharp
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
```

## चरण 3: हस्ताक्षर पंक्ति सम्मिलित करना

अब, का उपयोग करें `DocumentBuilder` अपने दस्तावेज़ में हस्ताक्षर पंक्ति सम्मिलित करने के लिए क्लास:

```csharp
SignatureLine signatureLine = builder.InsertSignatureLine(new SignatureLineOptions()).SignatureLine;
```

## चरण 4: दस्तावेज़ सहेजना

हस्ताक्षर लाइन डालने के बाद, दस्तावेज़ को सेव कर लें। हस्ताक्षर करने से पहले यह एक महत्वपूर्ण कदम है।

```csharp
doc.Save(dataDir + "SignDocuments.SignatureLine.docx");
```

## चरण 5: हस्ताक्षर विकल्प कॉन्फ़िगर करना

हस्ताक्षर प्रक्रिया के लिए विकल्प सेट करें। इसमें हस्ताक्षर लाइन आईडी और हस्ताक्षर के साथ प्रदर्शित करने के लिए वैकल्पिक छवि निर्दिष्ट करना शामिल है।

```csharp
SignOptions signOptions = new SignOptions
{
    SignatureLineId = signatureLine.Id,
    SignatureLineImage = File.ReadAllBytes(dataDir + "Enhanced Windows MetaFile.emf") // आपकी छवि तक का पथ
};
```

## चरण 6: प्रमाणपत्र लोड करना

दस्तावेज़ पर हस्ताक्षर करने के लिए आवश्यक प्रमाणपत्र फ़ाइल लोड करें:

```csharp
CertificateHolder certHolder = CertificateHolder.Create(dataDir + "your_certificate.pfx", "your_password"); // फ़ाइल नाम और पासवर्ड समायोजित करें
```

## चरण 7: दस्तावेज़ पर हस्ताक्षर करना

अंत में, दस्तावेज़ पर हस्ताक्षर करें `DigitalSignatureUtil` हस्ताक्षरित दस्तावेज़ को भविष्य में संदर्भ के लिए नए नाम से सहेजें।

```csharp
DigitalSignatureUtil.Sign(dataDir + "SignDocuments.SignatureLine.docx",
    dataDir + "SignDocuments.SignedDocument.docx", certHolder, signOptions);
```

## निष्कर्ष

बधाई हो! आपने सफलतापूर्वक एक Word दस्तावेज़ बना लिया है, उसमें हस्ताक्षर पंक्ति जोड़ दी है, और Aspose.Words for .NET का उपयोग करके उस पर डिजिटल हस्ताक्षर कर दिए हैं। यह शक्तिशाली टूल दस्तावेज़ स्वचालन को सरल बनाता है, यह सुनिश्चित करता है कि आपके अनुबंध और औपचारिक दस्तावेज़ सुरक्षित रूप से हस्ताक्षरित और प्रमाणित हों।

## अक्सर पूछे जाने वाले प्रश्न

### क्या मैं हस्ताक्षर लाइन के लिए अन्य छवि प्रारूपों का उपयोग कर सकता हूं?

हां, आप PNG, JPG और BMP सहित विभिन्न छवि प्रारूपों का उपयोग कर सकते हैं।

### क्या इसका उपयोग करना आवश्यक है? `.pfx` प्रमाण पत्र के लिए फाइल करें?

हाँ, एक `.pfx` फ़ाइल डिजिटल हस्ताक्षरों के लिए प्रमाणपत्रों और निजी कुंजियों को संग्रहीत करने के लिए एक मानक प्रारूप है।

### क्या मैं एक ही दस्तावेज़ में एकाधिक हस्ताक्षर पंक्तियाँ जोड़ सकता हूँ?

बिल्कुल! आप आवश्यकतानुसार प्रविष्टि चरण को दोहराकर कई हस्ताक्षर पंक्तियाँ सम्मिलित कर सकते हैं।

### यदि मेरे पास डिजिटल प्रमाणपत्र नहीं है तो क्या होगा?

आपको किसी विश्वसनीय प्रमाणपत्र प्राधिकारी से डिजिटल प्रमाणपत्र प्राप्त करना होगा या ओपनएसएसएल जैसे उपकरणों का उपयोग करके उसे तैयार करना होगा।

### मैं दस्तावेज़ में डिजिटल हस्ताक्षर का सत्यापन कैसे करूँ?

आप हस्ताक्षरित दस्तावेज़ को वर्ड में खोलकर तथा हस्ताक्षर विवरण की जांच करके उसकी प्रामाणिकता और अखंडता की पुष्टि करके डिजिटल हस्ताक्षर को सत्यापित कर सकते हैं।
