## परिचय

आज के डिजिटल दौर में, कुशल दस्तावेज़ प्रबंधन बेहद ज़रूरी है, खासकर PDF फ़ाइलों के मामले में। क्या आपने कभी खुद को किसी लंबी PDF फ़ाइल में अंतहीन स्क्रॉल करते हुए, किसी खास हिस्से को ढूँढ़ते हुए पाया है? निराशा होती है, है ना? यहीं पर बुकमार्क काम आते हैं! ये विषय-सूची की तरह काम करते हैं, जिससे पाठक आसानी से दस्तावेज़ों में नेविगेट कर सकते हैं। इस ट्यूटोरियल में, हम .NET के लिए Aspose.PDF का इस्तेमाल करके PDF फ़ाइल में चाइल्ड बुकमार्क जोड़ने का तरीका जानेंगे। इस गाइड के अंत तक, आप अपने PDF दस्तावेज़ों को बेहतर बना पाएँगे, उन्हें ज़्यादा उपयोगकर्ता-अनुकूल और व्यवस्थित बना पाएँगे।

## आवश्यक नामस्थान आयात करें

आपके शीर्ष पर `Program.cs` फ़ाइल में, आवश्यक नामस्थान आयात करें:

```csharp
using System;
using System.IO;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
```

## चरण 1: अपनी दस्तावेज़ निर्देशिका निर्दिष्ट करें

किसी भी PDF में हेरफेर करने से पहले, निर्दिष्ट करें कि आपके दस्तावेज़ कहाँ संग्रहीत हैं:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

प्रतिस्थापित करें `"YOUR DOCUMENT DIRECTORY"` आपकी पीडीएफ फाइल के वास्तविक पथ के साथ।

## चरण 2: पीडीएफ दस्तावेज़ खोलें

अब, उस पीडीएफ दस्तावेज़ को खोलें जिसके साथ आप काम करना चाहते हैं:

```csharp
Document pdfDocument = new Document(dataDir + "AddChildBookmark.pdf");
```

## चरण 3: पैरेंट बुकमार्क बनाएँ

इसके बाद, एक पैरेंट बुकमार्क बनाएं जो आपके चाइल्ड बुकमार्क के लिए मुख्य शीर्षक के रूप में काम करेगा:

```csharp
OutlineItemCollection parentBookmark = new OutlineItemCollection(pdfDocument.Outlines)
{
    Title = "Parent Outline",
    Italic = true,
    Bold = true
};
```

## चरण 4: चाइल्ड बुकमार्क बनाएँ

अब, पैरेंट बुकमार्क के अंतर्गत एक चाइल्ड बुकमार्क जोड़ें:

```csharp
OutlineItemCollection childBookmark = new OutlineItemCollection(pdfDocument.Outlines)
{
    Title = "Child Outline",
    Italic = true,
    Bold = true
};
```

## चरण 5: चाइल्ड बुकमार्क को पैरेंट बुकमार्क से लिंक करें

दोनों बुकमार्क बनाने के बाद, चाइल्ड बुकमार्क को पैरेंट बुकमार्क से लिंक करें:

```csharp
parentBookmark.Add(childBookmark);
```

## चरण 6: दस्तावेज़ में पैरेंट बुकमार्क जोड़ें

अब, मूल बुकमार्क (उसके चाइल्ड बुकमार्क के साथ) को दस्तावेज़ के आउटलाइन संग्रह में जोड़ें:

```csharp
pdfDocument.Outlines.Add(parentBookmark);
```

## चरण 7: दस्तावेज़ सहेजें

अंत में, पीडीएफ दस्तावेज़ में किए गए परिवर्तनों को सहेजें:

```csharp
dataDir = dataDir + "AddChildBookmark_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nChild bookmark added successfully.\nFile saved at " + dataDir);
```
