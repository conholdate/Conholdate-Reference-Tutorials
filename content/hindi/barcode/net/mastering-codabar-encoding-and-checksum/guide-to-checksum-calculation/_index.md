## परिचय

कोडाबार एक लोकप्रिय रैखिक बारकोड प्रतीक है जिसका उपयोग विभिन्न उद्योगों में लेबलिंग और पहचान में अपनी सरलता और दक्षता के लिए व्यापक रूप से किया जाता है। कोडाबार की एक महत्वपूर्ण विशेषता इसकी चेकसम गणना है, जो एन्कोड किए गए डेटा की सटीकता और अखंडता सुनिश्चित करने में मदद करती है। इस गाइड में, हम आपको .NET के लिए Aspose.BarCode का उपयोग करके विभिन्न चेकसम प्रकारों वाले कोडाबार बारकोड की गणना और जनरेट करने के चरणों से परिचित कराएँगे।

1. .NET के लिए Aspose.BarCode: सुनिश्चित करें कि आपके डेवलपमेंट परिवेश में यह लाइब्रेरी इंस्टॉल है। आप इसे यहाँ से डाउनलोड कर सकते हैं [यहाँ](https://releases.aspose.com/barcode/net/).

## आवश्यक नामस्थान आयात करना

Aspose.BarCode के साथ काम करने के लिए, अपनी C# फ़ाइल के शीर्ष पर आवश्यक नामस्थान आयात करके शुरू करें:

```csharp
using Aspose.BarCode.Generation;
```

## चरण 1: बारकोड जनरेटर को प्रारंभ करें

आपको बारकोड जेनरेटर को विशेष रूप से कोडाबार सिम्बोलॉजी के लिए इनिशियलाइज़ करना होगा। बदलना न भूलें `"Your Directory Path"` उस निर्देशिका पथ के साथ जहां आप बारकोड छवियों को सहेजना चाहते हैं।

```csharp
string path = "Your Directory Path";
System.Console.WriteLine("Codabar Checksum Examples:");

BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.Codabar, "-12345-");
```

## चरण 2: चेकसम के बिना कोडाबार बारकोड उत्पन्न करें

सबसे पहले, बिना किसी चेकसम के एक कोडाबार बारकोड बनाएँ। यह चेकसम विकल्प को पर सेट करके किया जाता है। `None`.

```csharp
gen.Parameters.Barcode.XDimension.Pixels = 2; // बार की चौड़ाई निर्धारित करें
gen.Parameters.Barcode.IsChecksumEnabled = EnableChecksum.Default; // कोई चेकसम नहीं
gen.Save($"{path}CodabarChecksumNone.png", BarCodeImageFormat.Png);
```

## चरण 3: Mod10 चेकसम के साथ कोडाबार बारकोड उत्पन्न करें

इसके बाद, हम एक कोडाबार बारकोड तैयार करेंगे जिसमें Mod10 चेकसम शामिल होगा, जो डेटा अखंडता को बढ़ाता है।

```csharp
gen.Parameters.Barcode.IsChecksumEnabled = EnableChecksum.Yes; // चेकसम सक्षम करें
gen.Parameters.Barcode.Codabar.CodabarChecksumMode = CodabarChecksumMode.Mod10; // सेट मॉड10
gen.Save($"{path}CodabarChecksumMod10.png", BarCodeImageFormat.Png);
```

## चरण 4: Mod16 चेकसम के साथ कोडाबार बारकोड उत्पन्न करें

अंत में, आइए एक कोडाबार बारकोड तैयार करें जो मॉड16 चेकसम का उपयोग करता है, जो उच्च सटीकता की मांग करने वाले अनुप्रयोगों के लिए उपयुक्त है।

```csharp
gen.Parameters.Barcode.IsChecksumEnabled = EnableChecksum.Yes; // चेकसम सक्षम करें
gen.Parameters.Barcode.Codabar.CodabarChecksumMode = CodabarChecksumMode.Mod16; // सेट मॉड16
gen.Save($"{path}CodabarChecksumMod16.png", BarCodeImageFormat.Png);
```
