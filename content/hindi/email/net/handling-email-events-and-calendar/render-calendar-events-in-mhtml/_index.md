## पैकेज आयात करें

आरंभ करने के लिए, अपने प्रोजेक्ट में आवश्यक नामस्थान शामिल करें। 

```csharp
using System;
using Aspose.Email;
using Aspose.Email.Mht;
```

अब, चलिए चरण-दर-चरण मार्गदर्शिका पर चलते हैं!

## चरण 1: कैलेंडर ईवेंट MSG फ़ाइल लोड करें

सबसे पहले, हम कैलेंडर ईवेंट वाली MSG फ़ाइल लोड करते हैं। यह चरण ज़रूरी है क्योंकि यह ईवेंट को MHTML फ़ॉर्मेट में रेंडर करने के लिए इनपुट का काम करता है।


```csharp
string dataDir = "Your Data Directory";
string fileName = "Meeting with Recurring Occurrences.msg";

// MSG फ़ाइल लोड करें
MailMessage msg = MailMessage.Load(dataDir + fileName);
```

- `dataDir`: वह निर्देशिका निर्दिष्ट करता है जहाँ आपकी MSG फ़ाइल संग्रहीत है।
- `fileName`: कैलेंडर ईवेंट फ़ाइल का नाम.
- `MailMessage.Load`: फ़ाइल को पढ़ता है और उसे लोड करता है `MailMessage` वस्तु।

## चरण 2: MHTML सहेजें विकल्प कॉन्फ़िगर करें

इसके बाद, हम कैलेंडर ईवेंट को MHTML फ़ॉर्मेट में रेंडर करने के विकल्पों को कॉन्फ़िगर करते हैं। इसमें अनुकूलन के लिए विशिष्ट फ़ॉर्मेट, हेडर और अन्य गुण सक्षम करना शामिल है।

```csharp
MhtSaveOptions options = new MhtSaveOptions
{
    MhtFormatOptions = MhtFormatOptions.WriteHeader | MhtFormatOptions.RenderCalendarEvent
};
```

- `MhtSaveOptions`: MHTML फ़ाइलों को सहेजने के लिए सेटिंग्स का प्रतिनिधित्व करता है।
- `MhtFormatOptions`: हेडर शामिल करने और कैलेंडर ईवेंट रेंडर करने जैसे विकल्पों को कॉन्फ़िगर करता है।

## चरण 3: प्रदर्शन टेम्पलेट्स को अनुकूलित करें

यहाँ, हम परिभाषित करते हैं कि विशिष्ट गुण, जैसे कि ईवेंट का प्रारंभ समय, आउटपुट में कैसे दिखाई देने चाहिए। यह चरण अत्यधिक अनुकूलन योग्य और पठनीय आउटपुट प्रदान करता है।


```csharp
if (options.FormatTemplates.ContainsKey(MhtTemplateName.Start))
    options.FormatTemplates[MhtTemplateName.Start] = @"<span class='headerLineTitle'>Start:</span><span class='headerLineText'>{0}</span><br/>";
else
    options.FormatTemplates.Add(MhtTemplateName.Start, @"<span class='headerLineTitle'>Start:</span><span class='headerLineText'>{0}</span><br/>");
```

- `MhtTemplateName.Start`: कैलेंडर ईवेंट की "प्रारंभ" संपत्ति को संदर्भित करता है।
- `options.FormatTemplates`: विशिष्ट गुणों के लिए प्रदर्शन टेम्पलेट को अनुकूलित करता है।

## चरण 4: कैलेंडर ईवेंट को MHTML के रूप में सहेजें

अंत में, कैलेंडर ईवेंट को कॉन्फ़िगर किए गए विकल्पों के साथ MHTML फ़ाइल में सहेजें।


```csharp
msg.Save(dataDir + "Meeting with Recurring Occurrences.mhtml", options);
```

- `msg.Save`: संदेश को निर्दिष्ट प्रारूप और स्थान में सहेजता है।
- `Meeting with Recurring Occurrences.mhtml`: आउटपुट फ़ाइल नाम.

## निष्कर्ष

.NET के लिए Aspose.Email का उपयोग करके कैलेंडर ईवेंट रेंडर करना कुशल और अत्यधिक अनुकूलन योग्य दोनों है। ऊपर दिए गए चरणों का पालन करके, आप कैलेंडर ईवेंट को अनुकूलित स्वरूपण के साथ एक MHTML फ़ाइल में आसानी से परिवर्तित कर सकते हैं।
