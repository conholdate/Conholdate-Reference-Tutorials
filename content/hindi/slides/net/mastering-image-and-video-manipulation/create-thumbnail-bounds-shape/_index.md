## आवश्यक नामस्थान आयात करें

Aspose.Slides की सुविधाओं का उपयोग करने के लिए, अपने प्रोजेक्ट की शुरुआत में आवश्यक नामस्थानों को आयात करके शुरू करें:

```csharp
using System.Drawing;
using System.Drawing.Imaging;
using Aspose.Slides;
```

## चरण 1: प्रेजेंटेशन क्लास को इंस्टैंशिएट करें

सबसे पहले, आपको आरंभीकरण करने की आवश्यकता है `Presentation` अपनी पावरपॉइंट फ़ाइल का प्रतिनिधित्व करने के लिए क्लास:

```csharp
string dataDir = "Your Documents Directory\\";
using (Presentation presentation = new Presentation(dataDir + "HelloWorld.pptx"))
{
    // अब आपका प्रेजेंटेशन ऑब्जेक्ट हेरफेर के लिए तैयार है।
}
```

का उपयोग `using` यह कथन यह सुनिश्चित करता है कि आपका काम पूरा हो जाने के बाद संसाधन उचित रूप से जारी कर दिए जाएं।

## चरण 2: आकार सीमाओं के साथ एक थंबनेल छवि बनाएँ

इसके बाद, आप अपनी प्रस्तुति में निर्दिष्ट सीमाओं के साथ एक आकृति की थंबनेल छवि बनाएंगे:

```csharp
using (Bitmap bitmap = presentation.Slides[0].Shapes[0].GetThumbnail(ShapeThumbnailBounds.Appearance, 1, 1))
{
    // बिटमैप में अब परिभाषित सीमा के भीतर थंबनेल छवि शामिल है।
}
```

इस अंश में, `ShapeThumbnailBounds.Appearance` यह निर्दिष्ट करता है कि आप आकृति की उपस्थिति सीमाएँ चाहते हैं। अपनी आउटपुट आवश्यकताओं के आधार पर चौड़ाई और ऊँचाई के लिए पैरामीटर (1, 1) को आवश्यकतानुसार समायोजित करें।

## चरण 3: थंबनेल छवि को डिस्क पर सहेजें

अंत में, उत्पन्न थंबनेल छवि को पसंदीदा प्रारूप में सहेजें, जैसे PNG:

```csharp
bitmap.Save(dataDir + "Shape_thumbnail_Bound_Shape_out.png", ImageFormat.Png);
```

यहां, आप अपनी परियोजना की आवश्यकताओं के अनुसार फ़ाइल नाम और प्रारूप को अनुकूलित कर सकते हैं।

बधाई हो! आपने Aspose.Slides for .NET का उपयोग करके एक आकृति के लिए सीमाओं सहित एक थंबनेल सफलतापूर्वक बना लिया है। यह प्रक्रिया सरल है और इसे आपके .NET अनुप्रयोगों में आसानी से एकीकृत किया जा सकता है।
