Selanjutnya, kita akan membahas sesuatu yang sedikit berbeda—bekerja dengan berkas ZIP. Jika Anda menangani beberapa dokumen LaTeX, seringkali lebih mudah untuk mengemasnya ke dalam format ZIP sebelum diproses.
