## Шаг 1: Настройте свои каталоги

Сначала укажите, где находится ваш пример файла Excel и куда вы хотите сохранить измененный вывод:

```csharp
// Исходный каталог
string sourceDir = "Your Document Directory";

// Выходной каталог
string outputDir = "Your Document Directory";
```

Заменять `"Your Document Directory"` с реальными путями. Это гарантирует, что код сможет корректно найти и сохранить файлы.

## Шаг 2: Загрузите образец файла Excel

Теперь давайте загрузим ваш пример файла Excel в программу:

```csharp
// Загрузите пример файла Excel, содержащего таблицу.
Workbook workbook = new Workbook(sourceDir + "sampleCreateSlicerToExcelTable.xlsx");
```

Мы используем `Workbook` Класс для загрузки нашего файла Excel. Убедитесь, что файл существует, чтобы избежать ошибок!

## Шаг 3: Откройте первый рабочий лист

Затем перейдите к нужному листу. Обычно это первый лист:

```csharp
// Откройте первый рабочий лист.
Worksheet worksheet = workbook.Worksheets[0];
```

Эта строка извлекает первый лист из книги. Если у вас несколько листов, измените индекс соответствующим образом.

## Шаг 4: Доступ к первой таблице на рабочем листе

Теперь найдите таблицу на листе, куда будет добавлен срез:

```csharp
// Доступ к первой таблице на рабочем листе.
ListObject table = worksheet.ListObjects[0];
```

Этот код извлекает первую таблицу на листе, позволяя работать с ней напрямую. Убедитесь, что таблица присутствует!

## Шаг 5: Добавьте слайсер

Таблица готова, давайте добавим срез! Он повышает интерактивность, выступая в роли графического фильтра для данных:

```csharp
int idx = worksheet.Slicers.Add(table, 0, "H5");
```

Здесь вы добавляете новый срез в таблицу и размещаете его в ячейке H5.

## Шаг 6: Доступ к слайсеру и изменение его свойств

Теперь, когда слайсер добавлен, вы можете настроить его свойства:

```csharp
Slicer slicer = worksheet.Slicers[idx];
slicer.Placement = PlacementType.FreeFloating;
slicer.RowHeightPixel = 50;
slicer.WidthPixel = 500;
slicer.Title = "Aspose";
slicer.AlternativeText = "Alternate Text";
slicer.IsPrintable = false;
slicer.IsLocked = false;
```

- Размещение: определяет, как срез взаимодействует с ячейками. `FreeFloating` позволяет осуществлять независимое передвижение.
- RowHeightPixel и WidthPixel: отрегулируйте размер среза для лучшей видимости.
- Название: Задает метку для слайсера.
- Альтернативный текст: содержит описание доступности.
- IsPrintable: контролирует, отображается ли срез в печатных версиях.
- IsLocked: определяет, могут ли пользователи перемещать или изменять размер слайсера.

## Шаг 7: Обновите слайсер

Чтобы изменения вступили в силу, обновите слайсер:

```csharp
// Обновите слайсер.
slicer.Refresh();
```

Эта строка применяет все ваши изменения, гарантируя, что слайсер отражает ваши обновления.

## Шаг 8: Сохраните рабочую книгу

Наконец, сохраните книгу с обновленными настройками слайсера:

```csharp
// Сохраните книгу в выходном формате XLSX.
workbook.Save(outputDir + "outputChangeSlicerProperties.xlsx", SaveFormat.Xlsx);
```

Теперь ваш измененный файл Excel будет сохранен в указанном выходном каталоге.
