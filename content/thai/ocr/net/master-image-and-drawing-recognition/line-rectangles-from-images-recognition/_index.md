## การแนะนำ

ยินดีต้อนรับสู่โลกของ Aspose.OCR สำหรับ .NET เครื่องมืออันทรงพลังที่ออกแบบมาเพื่อผสานรวม Optical Character Recognition (OCR) เข้ากับแอปพลิเคชัน .NET ของคุณ ไม่ว่าคุณจะเป็นนักพัฒนาที่มีประสบการณ์หรือมือใหม่ที่ใฝ่รู้ คู่มือนี้จะแนะนำขั้นตอนต่างๆ ในการดึงเส้นสี่เหลี่ยมจากข้อความที่รู้จำในรูปภาพ

## ข้อกำหนดเบื้องต้น

ก่อนที่คุณจะเริ่มต้น ให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

- ความรู้พื้นฐานเกี่ยวกับการพัฒนา C# และ .NET
- สภาพแวดล้อมการพัฒนาแบบบูรณาการ (IDE) เช่น Visual Studio
- ติดตั้งไลบรารี Aspose.OCR สำหรับ .NET แล้ว คุณสามารถดาวน์โหลดได้ [ที่นี่](https://releases-aspose.com/ocr/net/).
- ภาพตัวอย่างที่มีข้อความเพื่อการจดจำ

## เนมสเปซที่จำเป็น

ในการเริ่มต้น คุณจะต้องเพิ่มเนมสเปซที่จำเป็นลงในโปรเจ็กต์ของคุณ โดยใส่บรรทัดเหล่านี้ไว้ที่ด้านบนของไฟล์ C#:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

ปฏิบัติตามขั้นตอนเหล่านี้เพื่อดึงสี่เหลี่ยมสำหรับเส้นในภาพ OCR

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ

ระบุไดเรกทอรีที่ไฟล์รูปภาพของคุณตั้งอยู่:

```csharp
// กำหนดเส้นทางไปยังไดเรกทอรีเอกสารของคุณ
string dataDir = "Your Document Directory";
```

อย่าลืมเปลี่ยน `"Your Document Directory"` กับเส้นทางที่แท้จริง

## ขั้นตอนที่ 2: เริ่มต้น Aspose.OCR

สร้างอินสแตนซ์ของ `AsposeOcr` คลาสเพื่อเข้าถึงคุณลักษณะของมัน:

```csharp
// เริ่มต้น API ของ Aspose.OCR
AsposeOcr api = new AsposeOcr();
```

## ขั้นตอนที่ 3: ระบุเส้นทางของรูปภาพ

กำหนดเส้นทางแบบเต็มไปยังไฟล์รูปภาพที่คุณต้องการประมวลผล:

```csharp
// ระบุเส้นทางแบบเต็มไปยังรูปภาพ
string fullPath = dataDir + "sample.png";
```

## ขั้นตอนที่ 4: จดจำภาพและรับสี่เหลี่ยมเป็นเส้น

ตอนนี้คุณสามารถใช้ `GetRectangles` วิธีการแยกสี่เหลี่ยมของบรรทัดข้อความที่รู้จัก:

```csharp
// ดึงรูปสี่เหลี่ยมสำหรับเส้นในภาพที่ระบุ
List<Rectangle> lines = api.GetRectangles(fullPath, AreasType.LINES, false);
```

## ขั้นตอนที่ 5: ส่งออกผลลัพธ์

ในที่สุด ให้พิมพ์พิกัดของสี่เหลี่ยมเส้นที่ตรวจพบแต่ละเส้นไปยังคอนโซล:

```csharp
// แสดงพิกัดของรูปสี่เหลี่ยมที่ตรวจพบ
Console.WriteLine("Areas coordinates:");
lines.ForEach(a => Console.WriteLine($"x:{a.X} y:{a.Y} width:{a.Width} height:{a.Height}"));
```
