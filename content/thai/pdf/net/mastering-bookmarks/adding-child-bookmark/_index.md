## การแนะนำ

ในโลกดิจิทัลทุกวันนี้ การจัดการเอกสารอย่างมีประสิทธิภาพเป็นสิ่งสำคัญ โดยเฉพาะอย่างยิ่งเมื่อต้องจัดการกับไฟล์ PDF คุณเคยต้องเลื่อนดูไฟล์ PDF ยาวๆ ไม่รู้จบ เพื่อค้นหาเฉพาะส่วนที่ต้องการอย่างสิ้นหวังหรือไม่? น่าหงุดหงิดใช่ไหม? นี่แหละคือจุดที่บุ๊กมาร์กเข้ามามีบทบาท! บุ๊กมาร์กทำหน้าที่เหมือนสารบัญ ช่วยให้ผู้อ่านสามารถนำทางไปยังเอกสารต่างๆ ได้อย่างง่ายดาย ในบทช่วยสอนนี้ เราจะมาสำรวจวิธีการเพิ่มบุ๊กมาร์กย่อยลงในไฟล์ PDF โดยใช้ Aspose.PDF สำหรับ .NET เมื่ออ่านจบคู่มือนี้ คุณจะได้ปรับปรุงเอกสาร PDF ของคุณให้ใช้งานง่ายและเป็นระเบียบมากขึ้น

## ข้อกำหนดเบื้องต้น

ก่อนที่จะดำเนินการเพิ่มบุ๊กมาร์ก โปรดตรวจสอบให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

1. Aspose.PDF สำหรับ .NET: ดาวน์โหลดไลบรารีจาก [เว็บไซต์ Aspose](https://releases-aspose.com/pdf/net/).
2. Visual Studio: สภาพแวดล้อมการพัฒนาสำหรับการเขียนและทดสอบโค้ดของคุณ
3. ความรู้พื้นฐานเกี่ยวกับ C#: ความคุ้นเคยกับการเขียนโปรแกรม C# จะช่วยให้คุณเข้าใจชิ้นส่วนโค้ดได้

## สร้างโครงการใหม่

1. เปิด Visual Studio และสร้างโปรเจ็กต์ C# ใหม่ เลือกแอปพลิเคชันคอนโซลเพื่อความเรียบง่าย

## นำเข้าเนมสเปซที่จำเป็น

ที่ด้านบนของคุณ `Program.cs` ไฟล์นำเข้าเนมสเปซที่จำเป็น:

```csharp
using System;
using System.IO;
using Aspose.Pdf.Annotations;
using Aspose.Pdf;
```

## ขั้นตอนที่ 1: ระบุไดเรกทอรีเอกสารของคุณ

ก่อนที่จะจัดการไฟล์ PDF ใด ๆ โปรดระบุว่าเอกสารของคุณจัดเก็บอยู่ที่ไหน:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

แทนที่ `"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงไปยังไฟล์ PDF ของคุณ

## ขั้นตอนที่ 2: เปิดเอกสาร PDF

ตอนนี้เรามาเปิดเอกสาร PDF ที่คุณต้องการใช้งานกัน:

```csharp
Document pdfDocument = new Document(dataDir + "AddChildBookmark.pdf");
```

## ขั้นตอนที่ 3: สร้างบุ๊กมาร์กผู้ปกครอง

ขั้นตอนต่อไปคือสร้างบุ๊กมาร์กของผู้ปกครองที่จะทำหน้าที่เป็นหัวข้อหลักสำหรับบุ๊กมาร์กของบุตรหลานของคุณ:

```csharp
OutlineItemCollection parentBookmark = new OutlineItemCollection(pdfDocument.Outlines)
{
    Title = "Parent Outline",
    Italic = true,
    Bold = true
};
```

## ขั้นตอนที่ 4: สร้างบุ๊กมาร์กสำหรับเด็ก

ตอนนี้เรามาเพิ่มบุ๊กมาร์กย่อยใต้บุ๊กมาร์กหลักกัน:

```csharp
OutlineItemCollection childBookmark = new OutlineItemCollection(pdfDocument.Outlines)
{
    Title = "Child Outline",
    Italic = true,
    Bold = true
};
```

## ขั้นตอนที่ 5: เชื่อมโยงบุ๊กมาร์กของเด็กกับผู้ปกครอง

เมื่อสร้างบุ๊กมาร์กทั้งสองแล้ว ให้เชื่อมโยงบุ๊กมาร์กย่อยกับบุ๊กมาร์กหลัก:

```csharp
parentBookmark.Add(childBookmark);
```

## ขั้นตอนที่ 6: เพิ่มบุ๊กมาร์กหลักลงในเอกสาร

ตอนนี้ เพิ่มบุ๊กมาร์กของผู้ปกครอง (พร้อมกับลูกของบุ๊กมาร์ก) ลงในคอลเลกชันโครงร่างของเอกสาร:

```csharp
pdfDocument.Outlines.Add(parentBookmark);
```

## ขั้นตอนที่ 7: บันทึกเอกสาร

สุดท้ายบันทึกการเปลี่ยนแปลงที่ทำกับเอกสาร PDF:

```csharp
dataDir = dataDir + "AddChildBookmark_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nChild bookmark added successfully.\nFile saved at " + dataDir);
```
