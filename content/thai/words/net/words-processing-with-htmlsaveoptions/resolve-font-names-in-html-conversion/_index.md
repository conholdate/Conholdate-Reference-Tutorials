## การนำเข้าเนมสเปซที่จำเป็น

ก่อนเขียนโค้ด โปรดตรวจสอบให้แน่ใจว่าคุณได้นำเข้าเนมสเปซที่จำเป็นเข้าสู่โปรเจ็กต์ของคุณแล้ว ซึ่งเป็นสิ่งสำคัญสำหรับการเข้าถึงฟังก์ชันการทำงานของ Aspose.Words

using Aspose.Words;

## ขั้นตอนที่ 1: การตั้งค่าไดเรกทอรีเอกสาร

ขั้นแรก ให้ตั้งค่าเส้นทางไปยังไดเร็กทอรีเอกสารของคุณ ซึ่งเป็นตำแหน่งที่เอกสาร Word ของคุณตั้งอยู่และเป็นที่ที่คุณจะบันทึกผลลัพธ์ของคุณ

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "YOUR_DOCUMENT_DIRECTORY";
```

ที่นี่, `dataDir` เก็บเส้นทางไปยังไดเรกทอรีเอกสารของคุณ แทนที่ `"YOUR_DOCUMENT_DIRECTORY"` ด้วยเส้นทางจริงบนระบบของคุณ

## ขั้นตอนที่ 2: การโหลดเอกสาร Word

ต่อไป เราต้องโหลดเอกสาร Word ที่เราต้องการประมวลผล เอกสารนี้ควรมีฟอนต์ที่คุณต้องการแก้ไข

```csharp
Document doc = new Document(dataDir + "MissingFont.docx");
```

เราสร้าง `Document` วัตถุและโหลดเอกสาร Word ชื่อ "MissingFont.docx" จากของเรา `dataDir`-

## ขั้นตอนที่ 3: การกำหนดค่าตัวเลือกการบันทึก HTML

ตอนนี้เรามาตั้งค่าตัวเลือกสำหรับการบันทึกเอกสารเป็น HTML เพื่อให้แน่ใจว่าชื่อแบบอักษรได้รับการแก้ไขอย่างถูกต้อง

```csharp
HtmlSaveOptions saveOptions = new HtmlSaveOptions(SaveFormat.Html)
{
    PrettyFormat = true,
    ResolveFontNames = true
};
```

เราสร้างอินสแตนซ์ของ `HtmlSaveOptions` กับ `SaveFormat.Html`. การ `PrettyFormat` ตัวเลือกทำให้ผลลัพธ์ HTML อ่านง่ายขึ้น และ `ResolveFontNames` ช่วยให้แน่ใจว่าชื่อแบบอักษรได้รับการแก้ไขแล้ว

## ขั้นตอนที่ 4: บันทึกเอกสารเป็น HTML

สุดท้ายเราบันทึกเอกสารเป็นไฟล์ HTML โดยใช้ตัวเลือกการบันทึกที่กำหนดค่าไว้

```csharp
doc.Save(dataDir + "ResolvedFontNames.html", saveOptions);
```

เราเรียกว่า `Save` วิธีการบน `Document` วัตถุ โดยระบุเส้นทางเอาต์พุตและตัวเลือกการบันทึกที่เราตั้งค่าไว้ การดำเนินการนี้จะสร้างไฟล์ HTML ที่มีชื่อฟอนต์ที่ได้รับการแก้ไขแล้ว
