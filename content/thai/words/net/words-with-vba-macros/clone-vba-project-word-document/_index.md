3. ความรู้พื้นฐานเกี่ยวกับ C#: ความคุ้นเคยกับ C# จะช่วยให้คุณติดตามชิ้นส่วนโค้ดได้

## การนำเข้าเนมสเปซ

ในการเริ่มต้น คุณจะต้องนำเข้าเนมสเปซที่จำเป็นจาก Aspose.Words เนมสเปซเหล่านี้มีคลาสและเมธอดที่จำเป็นสำหรับการจัดการเอกสาร

วิธีการนำเข้ามีดังนี้:

```csharp
using Aspose.Words;
using Aspose.Words.Vba;
```

## ขั้นตอนที่ 1: การตั้งค่าไดเรกทอรีเอกสารของคุณ

ก่อนอื่น เราต้องกำหนดเส้นทางไปยังไดเรกทอรีเอกสารของคุณ นี่คือที่ที่คุณจะบันทึกเอกสาร Word ต้นทางและเอกสารใหม่

ตั้งค่าเส้นทางไปยังไดเร็กทอรีของคุณดังนี้:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

อย่าลืมเปลี่ยน `"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริงที่เก็บเอกสาร Word ของคุณ ไดเรกทอรีนี้จะทำหน้าที่เป็นพื้นที่ทำงานของเราสำหรับบทช่วยสอนนี้

## ขั้นตอนที่ 2: การโหลดเอกสาร Word

เมื่อตั้งค่าไดเรกทอรีเรียบร้อยแล้ว ก็ถึงเวลาโหลดเอกสาร Word ที่มีโปรเจกต์ VBA ที่คุณต้องการโคลน ขั้นตอนนี้สำคัญมากสำหรับการเข้าถึงโปรเจกต์ VBA ภายในเอกสาร

คุณสามารถโหลดเอกสารของคุณได้ดังนี้:

```csharp
Document doc = new Document(dataDir + "VBA project.docm");
```

โค้ดนี้จะโหลดเอกสาร Word ชื่อ "VBA project.docm" จากไดเร็กทอรีที่คุณระบุลงใน `doc` วัตถุ.

## ขั้นตอนที่ 3: การโคลนโครงการ VBA

ตอนนี้เราได้โหลดเอกสารต้นฉบับเรียบร้อยแล้ว มาโคลนโปรเจกต์ VBA ทั้งหมดกัน ซึ่งหมายความว่าต้องคัดลอกโมดูล การอ้างอิง และการตั้งค่าทั้งหมดจากเอกสารต้นฉบับไปยังเอกสารใหม่

นี่คือโค้ดที่จะทำสิ่งนี้:

```csharp
Document destDoc = new Document { VbaProject = doc.VbaProject.Clone() };
```

ในบรรทัดนี้เราสร้างเอกสารใหม่ `destDoc` และตั้งค่าโครงการ VBA ให้เป็นโคลนของโครงการ VBA จาก `doc`การดำเนินการนี้จะทำซ้ำเนื้อหา VBA ทั้งหมดจากเอกสารต้นฉบับลงในเอกสารใหม่

## ขั้นตอนที่ 4: การบันทึกเอกสารใหม่

เมื่อโคลนโครงการ VBA สำเร็จ ขั้นตอนสุดท้ายคือการบันทึกเอกสารใหม่ วิธีนี้ช่วยให้มั่นใจได้ว่าการเปลี่ยนแปลงทั้งหมดของคุณจะถูกเก็บรักษาไว้และเอกสารใหม่จะพร้อมใช้งาน

นี่คือโค้ดสำหรับบันทึกเอกสารใหม่ของคุณ:

```csharp
destDoc.Save(dataDir + "WorkingWithVba.CloneVbaProject.docm");
```

บรรทัดนี้จะบันทึกเอกสารใหม่พร้อมกับโครงการ VBA ที่โคลนเป็น "WorkingWithVba.CloneVbaProject.docm" ในไดเร็กทอรีที่คุณระบุ
