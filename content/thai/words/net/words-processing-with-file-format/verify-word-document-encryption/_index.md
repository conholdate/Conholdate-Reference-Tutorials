## การแนะนำ

คุณเคยเจอเอกสาร Word ที่เข้ารหัสแล้วสงสัยว่าจะตรวจสอบสถานะการเข้ารหัสด้วยโปรแกรมได้อย่างไรไหม ถ้าเคย คุณมาถูกที่แล้ว! ในบทช่วยสอนนี้ เราจะมาสำรวจวิธีการตรวจสอบสถานะการเข้ารหัสโดยใช้ไลบรารี Aspose.Words สำหรับ .NET ติดตามขั้นตอนการตั้งค่าและโค้ดเพื่อให้การตรวจสอบสถานะของคุณเป็นไปอย่างราบรื่น

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่มต้นเขียนโค้ด เรามาตรวจสอบให้แน่ใจว่าคุณมีทุกสิ่งที่คุณต้องการก่อน:

- Aspose.Words สำหรับไลบรารี .NET: ดาวน์โหลดจาก [ที่นี่](https://releases-aspose.com/words/net/).
- .NET Framework: ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง .NET Framework บนเครื่องของคุณแล้ว
- IDE: สภาพแวดล้อมการพัฒนาแบบบูรณาการ เช่น Visual Studio
- ความรู้พื้นฐานเกี่ยวกับ C#: ความคุ้นเคยกับ C# จะช่วยให้คุณทำตามบทช่วยสอนนี้ได้อย่างง่ายดาย

## ขั้นตอนที่ 1: นำเข้าเนมสเปซที่จำเป็น

ในการเริ่มต้น คุณจะต้องนำเข้าเนมสเปซที่จำเป็น เพิ่มบรรทัดต่อไปนี้ลงในโค้ดของคุณ:

```csharp
using Aspose.Words;
```

## ขั้นตอนที่ 2: กำหนดไดเรกทอรีเอกสาร

ต่อไป ให้ระบุเส้นทางไปยังไดเร็กทอรีที่เก็บเอกสารของคุณ แทนที่ `"YOUR DOCUMENT DIRECTORY"` พร้อมเส้นทางจริง:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## ขั้นตอนที่ 3: ตรวจจับรูปแบบไฟล์

ตอนนี้เราจะใช้ `DetectFileFormat` วิธีการจาก `FileFormatUtil` คลาสนี้ใช้เพื่อรวบรวมข้อมูลเกี่ยวกับรูปแบบไฟล์ สำหรับตัวอย่างนี้ เราถือว่าเอกสารที่เข้ารหัสชื่อ "Encrypted.docx" และอยู่ในไดเรกทอรีที่ระบุ:

```csharp
FileFormatInfo info = FileFormatUtil.DetectFileFormat(dataDir + "Encrypted.docx");
```

## ขั้นตอนที่ 4: ตรวจสอบว่าเอกสารได้รับการเข้ารหัสหรือไม่

เพื่อตรวจสอบว่าเอกสารได้รับการเข้ารหัสหรือไม่ เราสามารถใช้ `IsEncrypted` ทรัพย์สินของ `FileFormatInfo` วัตถุ คุณสมบัตินี้ส่งคืน `true` หากเอกสารถูกเข้ารหัสและ `false` มิฉะนั้น เราจะแสดงผลลัพธ์ในคอนโซล:

```csharp
Console.WriteLine($"Is the document encrypted? {info.IsEncrypted}");
```
