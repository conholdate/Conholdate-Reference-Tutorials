## แพ็คเกจนำเข้า

ในการใช้ Aspose.Email สำหรับ .NET คุณจะต้องนำเข้าเนมสเปซที่จำเป็นเข้าสู่โปรเจกต์ C# ของคุณ โดยเพิ่มบรรทัดต่อไปนี้ที่ด้านบนของโค้ด:

```csharp
using System;
using Aspose.Email;
```

มาถึงขั้นตอนที่สนุกแล้ว นั่นคือการแบ่งกระบวนการออกเป็นขั้นตอนที่จัดการได้ แต่ละขั้นตอนจะมีคำอธิบายโดยละเอียดเพื่อให้ง่ายต่อการปฏิบัติตาม

## ขั้นตอนที่ 1: ตั้งค่าเส้นทางไฟล์

ขั้นแรก คุณต้องมีไดเรกทอรีสำหรับบันทึกไฟล์ ICS ของคุณ เส้นทางนี้จะทำหน้าที่เป็นปลายทางสำหรับไฟล์ ICS ที่คุณแก้ไขแล้ว

```csharp
// เส้นทางไปยังไดเร็กทอรีไฟล์
string dataDir = "Your Data Directory";
```
 
การ `dataDir` ตัวแปรช่วยให้คุณจัดระเบียบไฟล์และรับรองว่าไฟล์ ICS จะถูกบันทึกไว้ในตำแหน่งที่ถูกต้อง แทนที่ `"Your Data Directory"` ด้วยเส้นทางที่ถูกต้องบนระบบของคุณ

## ขั้นตอนที่ 2: สร้างการนัดหมาย

ต่อไปสร้าง `Appointment` วัตถุ นี่แสดงถึงเหตุการณ์ในปฏิทินของคุณและประกอบด้วยคุณสมบัติต่างๆ เช่น ตำแหน่งที่ตั้ง หัวเรื่อง คำอธิบาย วันที่เริ่มต้น และวันที่สิ้นสุด

```csharp
string description = "Test Description";
Appointment app = new Appointment(
    "location", 
    "test appointment", 
    description, 
    DateTime.Today,
    DateTime.Today.AddDays(1), 
    "first@test.com", 
    "second@test.com"
);
```
 
- สถานที่ : ที่ที่จัดงาน  
- เรื่อง: ชื่อเรื่องสั้นๆ สำหรับกิจกรรมของคุณ  
- คำอธิบาย: รายละเอียดเพิ่มเติมเกี่ยวกับกิจกรรม  
- วันที่เริ่มต้นและสิ้นสุด: กำหนดระยะเวลาของกิจกรรม  
- ผู้เข้าร่วม: ระบุที่อยู่อีเมลของผู้ส่งและผู้รับ

## ขั้นตอนที่ 3: กำหนดตัวเลือกการบันทึก ICS

เพื่อปรับเปลี่ยน `ProdID`คุณจะต้องใช้ `IcsSaveOptions`ซึ่งจะทำให้คุณสามารถกำหนดค่าการตั้งค่าการบันทึกต่างๆ สำหรับไฟล์ ICS ได้

```csharp
IcsSaveOptions saveOptions = IcsSaveOptions.Default;
saveOptions.ProductId = "Your New ProdID"; // ปรับเปลี่ยน ProdID ตามต้องการ
```
 
การ `ProdID` ระบุซอฟต์แวร์ที่สร้างไฟล์ ICS การเปลี่ยนแปลงไฟล์สามารถช่วยสร้างแบรนด์ แก้จุดบกพร่อง หรือรับรองความเข้ากันได้กับแอปพลิเคชันเฉพาะ

## ขั้นตอนที่ 4: บันทึกไฟล์ ICS ที่แก้ไขแล้ว

สุดท้าย ให้บันทึกการนัดหมายที่อัปเดตลงในไฟล์ ICS โดยใช้ `Save` วิธี.

```csharp
// บันทึกการนัดหมายที่แก้ไขเป็นไฟล์ ICS
app.Save(dataDir + "ModifiedICSFile.ics", saveOptions);
```

ที่นี่เกิดอะไรขึ้น?  
การ `Save` วิธีนี้จะนำเส้นทางไฟล์และบันทึกตัวเลือกเป็นพารามิเตอร์ โดยจะสร้างไฟล์ ICS ด้วยไฟล์ที่คุณกำหนดเอง `ProdID`-
