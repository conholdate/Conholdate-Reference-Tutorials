## แพ็คเกจที่จำเป็นในการนำเข้า

ในการใช้ Aspose.Cells ให้เพิ่มเนมสเปซที่จำเป็นลงในไฟล์โค้ดของคุณ:

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ

ขั้นแรก ให้กำหนดเส้นทางสำหรับไดเรกทอรีเอกสารของคุณ นี่คือตำแหน่งที่จะเก็บไฟล์ Excel ของคุณ และตำแหน่งที่จะบันทึกไฟล์ผลลัพธ์หลังจากประมวลผล

```csharp
// เส้นทางไปยังไดเร็กทอรีเอกสาร
string dataDir = "Your Document Directory";
```

แทนที่ `"Your Document Directory"` พร้อมเส้นทางจริงไปยังไฟล์ Excel ของคุณ

## ขั้นตอนที่ 2: สร้างวัตถุเวิร์กบุ๊ก

ต่อไปสร้าง `Workbook` วัตถุที่จะแสดงไฟล์ Excel ของคุณ วัตถุนี้จะประกอบด้วยเวิร์กชีตทั้งหมดของคุณ

```csharp
// การสร้างอินสแตนซ์ของวัตถุเวิร์กบุ๊ก
Workbook workbook = new Workbook();
```

คุณสามารถโหลดเวิร์กบุ๊กที่มีอยู่ได้โดยระบุเส้นทางไฟล์หากคุณต้องการแก้ไขไฟล์ Excel ที่สร้างไว้แล้ว

## ขั้นตอนที่ 3: ล้างการแบ่งหน้าแนวนอนและแนวตั้ง

ทีนี้ มาล้างตัวแบ่งหน้ากัน ใน Excel คุณสามารถมีตัวแบ่งหน้าได้ทั้งแนวนอนและแนวตั้ง หากต้องการลบ ให้กำหนดเป้าหมาย `HorizontalPageBreaks` และ `VerticalPageBreaks` คอลเลกชันสำหรับเวิร์กชีตเฉพาะ:

```csharp
// การล้างตัวแบ่งหน้าทั้งหมด
workbook.Worksheets[0].HorizontalPageBreaks.Clear();
workbook.Worksheets[0].VerticalPageBreaks.Clear();
```

- `workbook.Worksheets[0]` กำหนดเป้าหมายไปที่แผ่นงานแรก
- `HorizontalPageBreaks.Clear()` ลบการแบ่งหน้าแนวนอนทั้งหมด
- `VerticalPageBreaks.Clear()` ลบการแบ่งหน้าแนวตั้งทั้งหมด

วิธีนี้จะช่วยให้คุณมีเวิร์กชีตที่สะอาดหมดจดโดยไม่มีการรบกวน

## ขั้นตอนที่ 4: บันทึกสมุดงาน

หลังจากล้างการแบ่งหน้าแล้ว ให้บันทึกการเปลี่ยนแปลงของคุณเพื่อสรุปเวิร์กบุ๊ก:

```csharp
// บันทึกไฟล์ Excel
workbook.Save(dataDir + "ClearAllPageBreaks_out.xls");
```

การดำเนินการนี้จะบันทึกเวิร์กบุ๊กลงในไดเร็กทอรีที่คุณระบุ โดยสร้างไฟล์ชื่อ `"ClearAllPageBreaks_out.xls"`คุณสามารถเปลี่ยนชื่อไฟล์เอาท์พุตได้ตามต้องการ
