## การแนะนำ

ในยุคดิจิทัล การจัดการและการทำงานร่วมกันในเอกสารมีความสำคัญอย่างยิ่งต่อเวิร์กโฟลว์ของเรา สเปรดชีต Excel ซึ่งมักเต็มไปด้วยข้อมูลและข้อมูลเชิงลึก มักมีคอมเมนต์ที่ให้บริบทหรือข้อเสนอแนะเพิ่มเติม ด้วย Aspose.Cells สำหรับ .NET การแยกและจัดการคอมเมนต์แบบเธรดจะราบรื่น บทช่วยสอนนี้จะแนะนำคุณตลอดขั้นตอนในการดึงคอมเมนต์แบบเธรดจากเวิร์กชีต Excel อย่างมีประสิทธิภาพ ไม่ว่าคุณจะมีประสบการณ์การเขียนโปรแกรมมากน้อยเพียงใด 

## ข้อกำหนดเบื้องต้น
ก่อนที่เราจะเริ่มเขียนโค้ด ให้แน่ใจว่าคุณมีสิ่งต่อไปนี้:

1. ความรู้พื้นฐานเกี่ยวกับ C#: ความคุ้นเคยกับ C# และ .NET Framework ถือเป็นสิ่งสำคัญ เนื่องจากตัวอย่างโค้ดของเราจะอยู่ในรูปแบบ C#
2. Visual Studio: ติดตั้ง Visual Studio บนเครื่องของคุณเพื่อรันโค้ด C#
3. Aspose.Cells สำหรับ .NET: ดาวน์โหลดและติดตั้งไลบรารี Aspose.Cells จาก [เว็บไซต์ Aspose](https://releases-aspose.com/cells/net/).
4. ตัวอย่างไฟล์ Excel: มีไฟล์ Excel ตัวอย่าง (เช่น `ThreadedCommentsSample.xlsx`) บันทึกไว้ในไดเร็กทอรีของคุณซึ่งประกอบด้วยข้อคิดเห็นแบบเธรดสำหรับการทดสอบ

## การนำเข้าแพ็คเกจที่จำเป็น
เพื่อใช้ประโยชน์จากฟีเจอร์อันทรงพลังของ Aspose.Cells คุณต้องรวมเนมสเปซที่จำเป็นไว้ในโปรเจ็กต์ C# ของคุณ โดยเพิ่มการประกาศต่อไปนี้ไว้ที่จุดเริ่มต้นของไฟล์ C#:

```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีแหล่งที่มา
ขั้นแรก ให้ระบุไดเรกทอรีที่ไฟล์ Excel ของคุณอยู่ ตรวจสอบให้แน่ใจว่าพาธตรงกับตำแหน่งของไฟล์ในระบบของคุณ

```csharp
// ไดเรกทอรีแหล่งที่มา
string sourceDir = "Your Document Directory";
```
แทนที่ `"Your Document Directory\"` พร้อมเส้นทางจริงไปยังไฟล์ Excel ของคุณ

## ขั้นตอนที่ 2: สร้างวัตถุเวิร์กบุ๊ก
ต่อไปสร้าง `Workbook` วัตถุที่จะโหลดและจัดการไฟล์ Excel ของคุณ

```csharp
// โหลดสมุดงาน
Workbook workbook = new Workbook(sourceDir + "ThreadedCommentsSample.xlsx");
```

## ขั้นตอนที่ 3: เข้าถึงแผ่นงาน
หลังจากโหลดเวิร์กบุ๊กแล้ว ให้เข้าถึงเวิร์กชีตที่มีคอมเมนต์แบบเธรด สำหรับตัวอย่างนี้ เราจะเข้าถึงเวิร์กชีตแรก

```csharp
// เข้าถึงแผ่นงานแรก
Worksheet worksheet = workbook.Worksheets[0];
```

## ขั้นตอนที่ 4: รับความคิดเห็นแบบเธรด
ตอนนี้ ให้ดึงความคิดเห็นแบบเธรดจากเซลล์ที่ระบุ เราจะกำหนดเป้าหมายไปที่เซลล์ "A1" ในตัวอย่างนี้

```csharp
// รับความคิดเห็นแบบเธรด
ThreadedCommentCollection threadedComments = worksheet.Comments.GetThreadedComments("A1");
```

## ขั้นตอนที่ 5: ทำซ้ำผ่านความคิดเห็น
โดยใช้การรวบรวมความคิดเห็นแบบเธรดในมือ ให้วนซ้ำผ่านความคิดเห็นแต่ละรายการเพื่อดึงข้อมูลที่เกี่ยวข้อง เช่น ข้อความความคิดเห็นและชื่อผู้เขียน

```csharp
// วนซ้ำผ่านความคิดเห็นแบบเธรดแต่ละข้อ
foreach (ThreadedComment comment in threadedComments)
{
    Console.WriteLine("Comment: " + comment.Notes);
    Console.WriteLine("Author: " + comment.Author.Name);
}
```

## ขั้นตอนที่ 6: ยืนยันการดำเนินการสำเร็จ
สุดท้ายเรามายืนยันกันว่าโปรแกรมของเราดำเนินการได้สำเร็จ

```csharp
Console.WriteLine("ReadThreadedComments executed successfully.");
```
