## Εισαγωγή

Τι συμβαίνει όμως όταν χρειάζεται να βελτιώσετε αυτά τα αρχεία MHT αλλάζοντας γραμματοσειρές; Χάρη στο Aspose.Email για .NET, αυτή η εργασία γίνεται παιχνιδάκι. Σε αυτό το σεμινάριο, θα σας καθοδηγήσουμε βήμα προς βήμα στη διαδικασία αλλαγής γραμματοσειρών κατά τη μετατροπή MHT. Είτε αναπτύσσετε μια εφαρμογή που χειρίζεται τη μορφοποίηση email είτε απλώς θέλετε να προσαρμόσετε έγγραφα για την επιχείρησή σας, αυτός ο οδηγός θα σας εξοπλίσει με τις γνώσεις που χρειάζεστε.

## Εισαγωγή πακέτων

Αρχικά, βεβαιωθείτε ότι το έργο σας έχει ρυθμιστεί ώστε να χρησιμοποιεί τους απαραίτητους χώρους ονομάτων. Θα πρέπει να συμπεριλάβετε τα ακόλουθα στην αρχή του αρχείου C#:

```csharp
using Aspose.Email;
using Aspose.Email.Mime;
using Aspose.Email.Tools;
```

Αυτά τα πακέτα θα σας δώσουν πρόσβαση στη λειτουργικότητα που απαιτείται για να εργαστείτε με αρχεία MHT και να τροποποιήσετε το περιεχόμενό τους.

Τώρα, ας αναλύσουμε τα βήματα που εμπλέκονται στην αλλαγή γραμματοσειρών κατά τη μετατροπή MHT.

## Βήμα 1: Φόρτωση του αρχείου MHT

Το πρώτο πράγμα που θα πρέπει να κάνετε είναι να φορτώσετε το αρχείο MHT σε ένα `MailMessage` αντικείμενο. Εδώ μπορείτε να αποκτήσετε πρόσβαση και να χειριστείτε το περιεχόμενό του.

```csharp
MailMessage message = MailMessage.Load("input.mht", new MhtmlLoadOptions());
```

Εξήγηση: Εδώ, `"input.mht"` είναι η διαδρομή προς το αρχείο MHT σας. Το `MhtmlLoadOptions()` σας επιτρέπει να διαμορφώσετε τον τρόπο φόρτωσης του αρχείου, για παράδειγμα, να χειρίζεστε τα συνημμένα ή τους συνδεδεμένους πόρους με διαφορετικό τρόπο.

## Βήμα 2: Επαναλάβετε μέσω εναλλακτικών προβολών

Τα αρχεία MHT συχνά έχουν πολλαπλές εναλλακτικές προβολές, ειδικά αν περιλαμβάνουν περιεχόμενο HTML. Πρέπει να κάνετε επανάληψη σε αυτές τις προβολές για να βρείτε αυτήν που θέλετε να τροποποιήσετε.

```csharp
foreach (var alternateView in message.AlternateViews)
{
    if (alternateView.ContentType.MediaType == "text/html")
    {
        var htmlView = (AlternateView)alternateView;
        var linkedResources = htmlView.LinkedResources;
```

Εξήγηση: Ελέγχετε κάθε `AlternateView` για να δείτε αν είναι τύπου HTML. Αν είναι, μπορείτε να αποκτήσετε πρόσβαση `LinkedResources`, όπου συνήθως αποθηκεύονται τυχόν γραμματοσειρές που είναι συνδεδεμένες στην HTML.

## Βήμα 3: Προσδιορισμός και Προσαρμογή Γραμματοσειρών

Τώρα που έχετε πρόσβαση στους συνδεδεμένους πόρους, μπορείτε να προσδιορίσετε ποιοι πόροι είναι γραμματοσειρές και να τους προσαρμόσετε όπως απαιτείται.

```csharp
foreach (var linkedResource in linkedResources)
{
    if (linkedResource.ContentType.MediaType == "application/x-font-ttf")
    {
        linkedResource.ContentType.Name = "Arial";  // Αλλαγή στην επιθυμητή γραμματοσειρά
        linkedResource.TransferEncoding = TransferEncoding.Base64;  // Βεβαιωθείτε ότι είναι σωστά κωδικοποιημένο
    }
}
```

Εξήγηση: Αυτός ο βρόχος ελέγχει εάν ο τύπος περιεχομένου του συνδεδεμένου πόρου είναι γραμματοσειρά TrueType. Εάν ταιριάζει, μπορείτε να αλλάξετε το όνομα της γραμματοσειράς σε αυτό που θέλετε (όπως "Arial" σε αυτό το παράδειγμα). `TransferEncoding` θα πρέπει επίσης να ρυθμιστεί ώστε να διασφαλίζεται ότι τα δεδομένα γραμματοσειράς κωδικοποιούνται σωστά κατά την αποθήκευση του εγγράφου.

## Βήμα 4: Αποθήκευση του ενημερωμένου αρχείου MHT

Αφού προσαρμόσετε τις γραμματοσειρές, ήρθε η ώρα να αποθηκεύσετε το τροποποιημένο αρχείο MHT. Θα πρέπει να βεβαιωθείτε ότι χρησιμοποιείτε τις σωστές επιλογές αποθήκευσης για να διατηρήσετε την ακεραιότητα του αρχείου σας.

```csharp
message.Save("output.mht", SaveOptions.DefaultMhtml);
```

Εξήγηση: Σε αυτήν τη γραμμή κώδικα, `"output.mht"` είναι το όνομα του αρχείου όπου θέλετε να αποθηκεύσετε το ενημερωμένο περιεχόμενο. Χρησιμοποιώντας `SaveOptions.DefaultMhtml` διασφαλίζει ότι το νέο αρχείο διατηρεί τη μορφή MHT.
