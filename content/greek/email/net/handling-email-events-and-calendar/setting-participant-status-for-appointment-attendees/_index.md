using Aspose.Email;

## Δημιουργία στιγμιότυπου ραντεβού

Τα ραντεβού στο Aspose.Email αντιπροσωπεύουν προγραμματισμένα συμβάντα, όπως συσκέψεις ή εργασίες. Δείτε πώς μπορείτε να δημιουργήσετε ένα:

```csharp
var appointment = new Appointment(
    "Conference Room 101", 
    DateTime.Now, 
    DateTime.Now.AddHours(1), 
    new MailAddress("organizer@example.com"),
    new MailAddressCollection { "attendee1@example.com", "attendee2@example.com" }
);
```

- Τοποθεσία: Καθορίζει πού θα πραγματοποιηθεί το ραντεβού.
- Ώρα έναρξης και ώρα λήξης: Ορίστε τη διάρκεια της συνάντησης.
- Διοργανωτής και Συμμετέχοντες: Ορίστε τους συμμετέχοντες και τους ρόλους τους.

## Προσθήκη συμμετεχόντων σε ραντεβού

Το Aspose.Email σάς επιτρέπει να διαχειρίζεστε μέσω προγραμματισμού τους συμμετέχοντες με τις διευθύνσεις email και τις καταστάσεις συμμετοχής τους.

```csharp
appointment.Attendees.Add(new MailAddress("john@example.com", "John Doe"));
appointment.Attendees.Add(new MailAddress("jane@example.com", "Jane Smith"));
```

## Διαχείριση Καταστάσεων Συμμετεχόντων

Ο `ParticipantStatus` Η ιδιότητα βοηθά στον προσδιορισμό του εάν ένας συμμετέχων έχει αποδεχτεί, απορρίψει ή αποδεχτεί διστακτικά μια πρόσκληση σε ραντεβού. Χρησιμοποιήστε τις ακόλουθες τιμές απαρίθμησης:

- Δεκτός
- Απορρίφθηκε
- Δοκιμαστικός

Παράδειγμα:

```csharp
appointment.Attendees[0].ParticipantStatus = AppointmentParticipantStatus.Accepted;
appointment.Attendees[1].ParticipantStatus = AppointmentParticipantStatus.Declined;
```

## Αποστολή ραντεβού ως προσκλήσεων σε συναντήσεις

Μόλις ετοιμαστεί το ραντεβού, μπορείτε να το στείλετε ως email πρόσκλησης:

```csharp
var msg = new MailMessage();
msg.From = "organizer@example.com";
msg.To = new MailAddressCollection { "john@example.com", "jane@example.com" };
msg.Subject = "Team Meeting";
msg.AlternateViews.Add(appointment.RequestApointment());

var client = new SmtpClient("smtp.example.com", 587, "username", "password");
client.Send(msg);
```
