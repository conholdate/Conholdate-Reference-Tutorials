## Εισαγωγή χώρων ονομάτων

Ξεκινήστε το έργο .NET εισάγοντας τους απαραίτητους χώρους ονομάτων για να αποκτήσετε πρόσβαση στις λειτουργίες του Aspose.Zip:

```csharp
using System;
using Aspose.Zip.Tar;
```

## Βήμα 1: Ορίστε τον κατάλογο εγγράφων σας

Ξεκινήστε ορίζοντας τον κατάλογο όπου βρίσκονται τα αρχεία σας. Αυτός θα αναφέρεται σε όλη τη διαδικασία συμπίεσης.

```csharp
string dataDir = "Your Document Directory";
```

## Βήμα 2: Δημιουργία αρχείου TarGz

Τώρα, ας προχωρήσουμε στη δημιουργία ενός αρχείου TarGz με το Aspose.Zip για .NET ακολουθώντας αυτά τα δευτερεύοντα βήματα:

### Βήμα 2.1: Αρχικοποίηση του TarArchive

Αρχικά, αρχικοποιήστε ένα `TarArchive` αντικείμενο για εργασία με:

```csharp
using (TarArchive archive = new TarArchive())
{
    // Προσθέστε αρχεία στο αρχείο όπως περιγράφεται στα επόμενα βήματα
}
```

### Βήμα 2.2: Προσθήκη καταχωρίσεων

Στη συνέχεια, προσθέστε τα αρχεία που θέλετε να συμπιέσετε στο αρχείο. Ακολουθεί ένα παράδειγμα με τα αρχεία `"alice29.txt"` και `"lcet10.txt"`:

```csharp
archive.CreateEntry("alice29.txt", dataDir + "alice29.txt");
archive.CreateEntry("lcet10.txt", dataDir + "lcet10.txt");
```

### Βήμα 2.3: Αποθήκευση ως Gzipped Tar

Τέλος, αποθηκεύστε το αρχείο που δημιουργήθηκε σε μορφή TarGz χρησιμοποιώντας το `SaveGzipped` μέθοδος:

```csharp
archive.SaveGzipped(dataDir + "archive.tar.gz");
```
