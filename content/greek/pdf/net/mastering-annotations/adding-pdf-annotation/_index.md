## Εισαγωγή απαραίτητων πακέτων

Πριν από την κωδικοποίηση, βεβαιωθείτε ότι έχετε εισαγάγει τους απαιτούμενους χώρους ονομάτων:

using Aspose.Pdf;

Αυτοί οι χώροι ονομάτων παρέχουν τις κλάσεις και τις μεθόδους που είναι απαραίτητες για τον χειρισμό και τη σχολιασμό PDF.

## Βήμα 1: Φόρτωση του εγγράφου PDF

Ξεκινήστε φορτώνοντας το έγγραφο PDF στο σημείο όπου θέλετε να προσθέσετε σχολιασμούς PDF.

```csharp
// Καθορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας.
string dataDir = "YOUR DATA DIRECTORY";
// Φόρτωση του εγγράφου PDF
Document pdfDocument = new Document(dataDir + "AddAnnotation.pdf");
```

Αυτό το απόσπασμα κώδικα ορίζει τον κατάλογο για το αρχείο PDF σας και το φορτώνει σε ένα `Document` αντικείμενο, επιτρέποντας περαιτέρω τροποποιήσεις.

## Βήμα 2: Δημιουργήστε μια σχολίαση

Στη συνέχεια, θα δημιουργήσουμε ένα `TextAnnotation`, ιδανικό για την προσθήκη σχολίων ή σημειώσεων.

```csharp
// Δημιουργία σχολίου κειμένου
TextAnnotation textAnnotation = new TextAnnotation(pdfDocument.Pages[1], new Aspose.Pdf.Rectangle(200, 400, 400, 600))
{
    Title = "Sample Annotation Title",
    Subject = "Sample Subject",
    Contents = "Sample contents for the annotation",
    Open = true,
    Icon = TextIcon.Key
};
```

- Τοποθεσία και Μέγεθος: Το `Rectangle` Η κλάση ορίζει τη θέση και τις διαστάσεις της σχολίασης στη σελίδα.
- Ιδιότητες: Μπορείτε να ορίσετε τον τίτλο, το θέμα και το περιεχόμενο της σχολίασης. `Open` Η ιδιότητα καθορίζει εάν η σχολίαση εμφανίζεται ανοιχτή από προεπιλογή.
- Εικονίδιο: Το `TextIcon.Key` προσθέτει ένα οπτικό στοιχείο στη σχολίαση.

## Βήμα 3: Προσαρμόστε την εμφάνιση του σχολιασμού

Βελτιώστε την ορατότητα του σχολιασμού προσαρμόζοντας την εμφάνισή του.

```csharp
// Προσαρμόστε το περίγραμμα του σχολιασμού
Border border = new Border(textAnnotation)
{
    Width = 5,
    Dash = new Dash(1, 1)
};
textAnnotation.Border = border;
textAnnotation.Rect = new Aspose.Pdf.Rectangle(200, 400, 400, 600);
```

- Περίγραμμα: Δημιουργήστε ένα `Border` αντικείμενο, ορίζοντας το πλάτος και το στυλ του (με διακεκομμένη γραμμή σε αυτήν την περίπτωση) για βελτιωμένη ορατότητα.

## Βήμα 4: Προσθέστε την σχολίαση στη σελίδα PDF

Τώρα, ήρθε η ώρα να προσθέσετε τη σχολίαση στη σελίδα PDF σας.

```csharp
// Προσθήκη της σχολίασης στη συλλογή σχολίων της σελίδας
pdfDocument.Pages[1].Annotations.Add(textAnnotation);
```

Αυτή η γραμμή προσθέτει τη νέα σας σχολίαση στην πρώτη σελίδα του PDF, ενσωματώνοντάς την στο έγγραφο.

## Βήμα 5: Αποθηκεύστε το ενημερωμένο έγγραφο PDF

Τέλος, αποθηκεύστε το έγγραφο για να διατηρήσετε τις αλλαγές σας.

```csharp
// Αποθήκευση του ενημερωμένου εγγράφου PDF
dataDir = dataDir + "AddAnnotation_out.pdf";
pdfDocument.Save(dataDir);
Console.WriteLine("\nAnnotation added successfully.\nFile saved at " + dataDir);
```

Αυτός ο κώδικας αποθηκεύει το τροποποιημένο έγγραφο ως `AddAnnotation_out.pdf`, διατηρώντας το αρχικό αρχείο και επιβεβαιώνοντας την επιτυχή προσθήκη της σχολίασης.
