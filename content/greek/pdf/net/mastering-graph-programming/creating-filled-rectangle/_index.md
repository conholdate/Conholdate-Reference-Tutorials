## Εισαγωγή

Θέλατε ποτέ να δημιουργήσετε οπτικά εντυπωσιακά PDF μέσω προγραμματισμού; Αν ναι, τότε βρίσκεστε στο σωστό μέρος! Σε αυτό το σεμινάριο, θα εξερευνήσουμε το Aspose.PDF για .NET, μια ισχυρή βιβλιοθήκη που απλοποιεί τον χειρισμό εγγράφων PDF. Σήμερα, θα επικεντρωθούμε στη δημιουργία ενός γεμισμένου ορθογωνίου μέσα σε ένα αρχείο PDF. Είτε είστε έμπειρος προγραμματιστής είτε μόλις ξεκινάτε, αυτός ο οδηγός θα σας καθοδηγήσει σε κάθε βήμα με φιλικό και ενδιαφέρον τρόπο. Πάρτε λοιπόν το καπέλο σας στον προγραμματισμό και ας ξεκινήσουμε!

## Προαπαιτούμενα

Πριν εμβαθύνουμε στον κώδικα, βεβαιωθείτε ότι έχετε τα εξής:

1. Visual Studio: Εγκαταστήστε το Visual Studio στον υπολογιστή σας, καθώς είναι ένα εξαιρετικό IDE για ανάπτυξη .NET.
2. Aspose.PDF για .NET: Λήψη και εγκατάσταση της βιβλιοθήκης Aspose.PDF από [εδώ](https://releases.aspose.com/pdf/net/).
3. Βασικές γνώσεις C#: Η εξοικείωση με τον προγραμματισμό C# θα σας βοηθήσει να κατανοήσετε καλύτερα τα αποσπάσματα κώδικα.

## Βήμα 1: Δημιουργία νέου έργου

1. Ανοίξτε το Visual Studio και δημιουργήστε ένα νέο έργο εφαρμογής κονσόλας.
2. Ονομάστε το έργο σας κατάλληλα.

## Βήμα 2: Προσθήκη αναφοράς Aspose.PDF

1. Κάντε δεξί κλικ στο έργο σας στην Εξερεύνηση λύσεων.
2. Επιλέξτε Διαχείριση πακέτων NuGet.
3. Αναζητήστε το Aspose.PDF και εγκαταστήστε την πιο πρόσφατη έκδοση.

```csharp
using System.IO;
using System;
using Aspose.Pdf;
```

Τώρα που έχουμε όλα ρυθμισμένα, ας περάσουμε στον κώδικα!

## Βήμα 3: Ρύθμιση του καταλόγου εγγράφων σας

Καθορίστε τη διαδρομή όπου θα αποθηκευτεί το PDF σας:

```csharp
// Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή στον υπολογιστή σας όπου θέλετε να αποθηκεύσετε το PDF.

## Βήμα 4: Δημιουργία μιας παρουσίας εγγράφου

Αρχικοποίηση ενός νέου εγγράφου PDF:

```csharp
// Δημιουργία στιγμιότυπου εγγράφου
Document doc = new Document();
```

## Βήμα 5: Προσθήκη σελίδας στο έγγραφο

Κάθε PDF χρειάζεται τουλάχιστον μία σελίδα. Ας προσθέσουμε μία:

```csharp
// Προσθήκη σελίδας σε συλλογή σελίδων αρχείου PDF
Page page = doc.Pages.Add();
```

## Βήμα 6: Δημιουργία μιας παρουσίας γραφήματος

ΕΝΑ `Graph` Η instance λειτουργεί ως καμβάς για τη σχεδίαση σχημάτων:

```csharp
// Δημιουργία στιγμιότυπου γραφήματος
Aspose.Pdf.Drawing.Graph graph = new Aspose.Pdf.Drawing.Graph(100.0, 400.0);
```

## Βήμα 7: Προσθήκη του γραφήματος στη σελίδα

Επισυνάψτε το γράφημα στη σελίδα:

```csharp
// Προσθήκη αντικειμένου γραφήματος στη συλλογή παραγράφων της παρουσίας σελίδας
page.Paragraphs.Add(graph);
```

## Βήμα 8: Δημιουργήστε μια παρουσία ορθογωνίου

Ορίστε τη θέση και το μέγεθος του ορθογωνίου:

```csharp
// Δημιουργία αντιγράφου ορθογωνίου
Aspose.Pdf.Drawing.Rectangle rect = new Aspose.Pdf.Drawing.Rectangle(100, 100, 200, 120);
```

## Βήμα 9: Καθορίστε το χρώμα γεμίσματος

Επιλέξτε ένα χρώμα για το ορθογώνιό σας. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε κόκκινο:

```csharp
// Καθορισμός χρώματος γεμίσματος για το αντικείμενο γραφήματος
rect.GraphInfo.FillColor = Aspose.Pdf.Color.Red;
```

## Βήμα 10: Προσθέστε το ορθογώνιο στο γράφημα

Προσθέστε το ορθογώνιο στο γράφημα:

```csharp
// Προσθήκη αντικειμένου ορθογωνίου σε συλλογή σχημάτων του αντικειμένου Γράφημα
graph.Shapes.Add(rect);
```

## Βήμα 11: Αποθήκευση του εγγράφου PDF

Τέλος, αποθηκεύστε το έγγραφό σας στον καθορισμένο κατάλογο:

```csharp
dataDir = dataDir + "CreateFilledRectangle_out.pdf";
// Αποθήκευση αρχείου PDF
doc.Save(dataDir);
```

## Βήμα 12: Μήνυμα επιβεβαίωσης

Εκτυπώστε ένα μήνυμα επιβεβαίωσης που υποδεικνύει την επιτυχία:

```csharp
Console.WriteLine("\nFilled rectangle object created successfully.\nFile saved at " + dataDir);
```
