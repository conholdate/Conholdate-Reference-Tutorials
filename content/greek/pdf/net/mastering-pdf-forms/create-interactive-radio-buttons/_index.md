## Εισαγωγή

Τα διαδραστικά PDF μπορούν να βελτιώσουν σημαντικά την αλληλεπίδραση των χρηστών, ειδικά όσον αφορά τις φόρμες. Ένα από τα πιο αποτελεσματικά διαδραστικά στοιχεία είναι το κουμπί επιλογής, το οποίο επιτρέπει στους χρήστες να επιλέξουν μία επιλογή από ένα σύνολο. Σε αυτό το σεμινάριο, θα δούμε τα βήματα για τη δημιουργία κουμπιών επιλογής σε ένα έγγραφο PDF χρησιμοποιώντας το Aspose.PDF για .NET. Είτε είστε έμπειρος προγραμματιστής είτε αρχάριος, αυτός ο οδηγός θα σας βοηθήσει να κατανοήσετε κάθε μέρος του κώδικα.

Τώρα που το περιβάλλον σας έχει ρυθμιστεί, ας εμβαθύνουμε στον κώδικα.

## Βήμα 1: Ορίστε τον κατάλογο εγγράφων σας

Καθορίστε τον κατάλογο όπου θα αποθηκευτεί το PDF σας:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY"; // Αντικαταστήστε με την πραγματική σας διαδρομή
```

## Βήμα 2: Δημιουργία αντικειμένου εγγράφου

Δημιουργήστε μια παρουσία του `Document` τάξη:

```csharp
Document pdfDocument = new Document();
```

## Βήμα 3: Προσθήκη σελίδας στο PDF

Προσθήκη νέας σελίδας στο έγγραφο PDF:

```csharp
pdfDocument.Pages.Add();
```

## Βήμα 4: Δημιουργήστε το πεδίο του κουμπιού επιλογής

Δημιουργήστε ένα υπόδειγμα `RadioButtonField` αντικείμενο για την πρώτη σελίδα:

```csharp
RadioButtonField radio = new RadioButtonField(pdfDocument.Pages[1]);
```

## Βήμα 5: Προσθήκη επιλογών στο κουμπί επιλογής

Ορίστε τις επιλογές για το κουμπί επιλογής σας:

```csharp
radio.AddOption("Option 1", new Rectangle(0, 0, 20, 20));
radio.AddOption("Option 2", new Rectangle(0, 30, 20, 20));
```

Αυτό το παράδειγμα προσθέτει δύο επιλογές: "Επιλογή 1" και "Επιλογή 2". `Rectangle` Το αντικείμενο καθορίζει τη θέση και το μέγεθος κάθε επιλογής.

## Βήμα 6: Προσθέστε το κουμπί επιλογής στη φόρμα εγγράφου

Ενσωματώστε το κουμπί επιλογής στη φόρμα PDF:

```csharp
pdfDocument.Form.Add(radio);
```

## Βήμα 7: Αποθήκευση του εγγράφου PDF

Αποθηκεύστε το έγγραφο PDF στον καθορισμένο κατάλογο:

```csharp
dataDir = dataDir + "RadioButton_out.pdf";
pdfDocument.Save(dataDir);
```
