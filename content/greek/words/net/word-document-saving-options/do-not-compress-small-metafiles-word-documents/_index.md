## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Αρχικά, καθορίστε τον κατάλογο όπου θα αποθηκευτεί το έγγραφό σας. Η σωστή διαχείριση των διαδρομών αρχείων είναι απαραίτητη.

```csharp
// Διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "YOUR DOCUMENTS DIRECTORY";
```

Αντικαθιστώ `"YOUR DOCUMENTS DIRECTORY"` με την πραγματική διαδρομή όπου θέλετε να αποθηκεύσετε το έγγραφό σας.

## Βήμα 2: Δημιουργία νέου εγγράφου

Στη συνέχεια, θα δημιουργήσουμε ένα νέο έγγραφο και θα προσθέσουμε κάποιο περιεχόμενο χρησιμοποιώντας ένα εργαλείο δημιουργίας εγγράφων.

```csharp
// Δημιουργήστε ένα νέο έγγραφο
Document doc = new Document();
DocumentBuilder builder = new DocumentBuilder(doc);
builder.Writeln("Text added to a document.");
```

Εδώ, ένα `Document` το αντικείμενο αρχικοποιείται και το `DocumentBuilder` χρησιμοποιείται για την εισαγωγή κειμένου. Το `Writeln` Η μέθοδος προσθέτει μια γραμμή κειμένου στο έγγραφο.

## Βήμα 3: Ρύθμιση παραμέτρων επιλογών αποθήκευσης

Τώρα, ρυθμίστε τις επιλογές αποθήκευσης για να χρησιμοποιήσετε τη λειτουργία "Μην συμπιέζετε μικρά μετααρχεία" με το `DocSaveOptions` τάξη.

```csharp
// Ρύθμιση παραμέτρων επιλογών αποθήκευσης με τη λειτουργία "Μην συμπιέζετε μικρά μετααρχεία"
DocSaveOptions saveOptions = new DocSaveOptions {
    Compliance = PdfCompliance.PdfA1a
};
```

Αυτό το βήμα δημιουργεί μια παρουσία του `DocSaveOptions` και ορίζει το `Compliance` ιδιοκτησία σε `PdfCompliance.PdfA1a`, διασφαλίζοντας ότι το έγγραφο συμμορφώνεται με το πρότυπο PDF/A-1a.

## Βήμα 4: Αποθήκευση του εγγράφου

Τέλος, αποθηκεύστε το έγγραφο χρησιμοποιώντας τις διαμορφωμένες επιλογές, διασφαλίζοντας ότι τα μικρά μετααρχεία δεν συμπιέζονται.

```csharp
// Αποθήκευση του εγγράφου με τις καθορισμένες επιλογές
doc.Save(dataDir + "DocumentWithDoNotCompressMetafiles.pdf", saveOptions);
```

Σε αυτή τη γραμμή, το `Save` μέθοδος του `Document` Η κλάση καλείται για την αποθήκευση του εγγράφου. Η διαδρομή συνδυάζει τον κατάλογό σας και το επιθυμητό όνομα αρχείου "DocumentWithDoNotCompressMetafiles.pdf".
