## Φόρτωση του εγγράφου του Word

Το πρώτο βήμα για την τροποποίηση ενός εγγράφου του Word είναι η φόρτωσή του στην εφαρμογή σας. Δείτε πώς μπορείτε να φορτώσετε ένα έγγραφο χρησιμοποιώντας το Aspose.Words για .NET:

### Βήμα 1: Ορίστε τη διαδρομή αρχείου

Πρέπει να ορίσετε τη διαδρομή αρχείου του εγγράφου του Word. Για αυτό το παράδειγμα, θα χρησιμοποιήσουμε το έγγραφο `Properties.docx`.

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
Document doc = new Document(dataDir + "Properties.docx");
```

Βεβαιωθείτε ότι θα αντικαταστήσετε `"YOUR DOCUMENT DIRECTORY"` με τον πραγματικό κατάλογο όπου είναι αποθηκευμένο το έγγραφό σας.

## Πρόσβαση και κατάργηση ιδιοτήτων προσαρμοσμένου εγγράφου

Μόλις φορτωθεί το έγγραφο στην εφαρμογή σας, μπορείτε να αποκτήσετε πρόσβαση στις προσαρμοσμένες ιδιότητές του και να τις καταργήσετε. Δείτε πώς μπορείτε να χειριστείτε αυτήν την εργασία:

### Βήμα 2: Ανάκτηση των ιδιοτήτων του προσαρμοσμένου εγγράφου

Αποκτήστε πρόσβαση στις προσαρμοσμένες ιδιότητες του φορτωμένου εγγράφου χρησιμοποιώντας το `CustomDocumentProperties` ιδιότητα. Αυτό σας επιτρέπει να διαχειρίζεστε και να τροποποιείτε τις ιδιότητες του εγγράφου μέσω προγραμματισμού.

```csharp
var customProperties = doc.CustomDocumentProperties;
```

### Βήμα 3: Αφαίρεση συγκεκριμένων ιδιοτήτων

Αν χρειάζεται να καταργήσετε μια προσαρμοσμένη ιδιότητα, απλώς καθορίστε το όνομα της ιδιότητας. Για παράδειγμα, ας υποθέσουμε ότι θέλετε να καταργήσετε την ιδιότητα που ονομάζεται `"Authorized Date"`. Ορίστε ο κώδικας για αυτό:

```csharp
customProperties.Remove("Authorized Date");
```

Καλώντας το `Remove` χρησιμοποιώντας τη μέθοδο και μεταβιβάζοντας το όνομα της ιδιότητας, μπορείτε εύκολα να διαγράψετε τυχόν περιττές ή παρωχημένες ιδιότητες.

## Αποθήκευση του τροποποιημένου εγγράφου

Μετά την κατάργηση των προσαρμοσμένων ιδιοτήτων, το τελευταίο βήμα είναι η αποθήκευση του τροποποιημένου εγγράφου. Αυτό διασφαλίζει ότι εφαρμόζονται όλες οι αλλαγές, συμπεριλαμβανομένης της κατάργησης των προσαρμοσμένων ιδιοτήτων.

### Βήμα 4: Ορίστε τη διαδρομή αποθήκευσης

Καθορίστε τη διαδρομή όπου θέλετε να αποθηκεύσετε το τροποποιημένο έγγραφο. Αυτή είναι η τοποθεσία όπου θα αποθηκευτεί το νέο αρχείο Word.

```csharp
string savePath = dataDir + "ModifiedProperties.docx";
```

### Βήμα 5: Αποθήκευση του εγγράφου

Τέλος, χρησιμοποιήστε το `Save` μέθοδος για την αποθήκευση του εγγράφου στην καθορισμένη διαδρομή:

```csharp
doc.Save(savePath);
```

Αυτό θα αποθηκεύσει το έγγραφο χωρίς τις προσαρμοσμένες ιδιότητες, διασφαλίζοντας ότι οι αλλαγές θα παραμείνουν.
