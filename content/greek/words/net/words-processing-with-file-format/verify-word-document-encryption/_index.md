## Εισαγωγή

Έχετε ποτέ συναντήσει ένα κρυπτογραφημένο έγγραφο του Word και αναρωτηθήκατε πώς να επαληθεύσετε την κατάσταση κρυπτογράφησής του μέσω προγραμματισμού; Αν ναι, βρίσκεστε στο σωστό μέρος! Σε αυτό το σεμινάριο, θα εξερευνήσουμε πώς να το πετύχετε αυτό χρησιμοποιώντας τη βιβλιοθήκη Aspose.Words για .NET. Ακολουθήστε μας καθώς σας καθοδηγούμε στη ρύθμιση και τον κώδικα για να ολοκληρωθεί ομαλά η επαλήθευσή σας.

Πριν προχωρήσουμε στον κώδικα, ας βεβαιωθούμε ότι έχετε όλα όσα χρειάζεστε:

- Aspose.Words για τη βιβλιοθήκη .NET: Κατεβάστε το από [εδώ](https://releases.aspose.com/words/net/).

- Βασικές γνώσεις C#: Η εξοικείωση με την C# θα σας βοηθήσει να παρακολουθήσετε εύκολα αυτό το σεμινάριο.

## Βήμα 1: Εισαγωγή απαιτούμενων χώρων ονομάτων

Για να ξεκινήσετε, θα χρειαστεί να εισαγάγετε τους απαραίτητους χώρους ονομάτων. Προσθέστε την ακόλουθη γραμμή στον κώδικά σας:

```csharp
using Aspose.Words;
```

## Βήμα 2: Ορισμός του καταλόγου εγγράφων

Στη συνέχεια, καθορίστε τη διαδρομή προς τον κατάλογο όπου είναι αποθηκευμένα τα έγγραφά σας. Αντικατάσταση `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Βήμα 3: Εντοπισμός της μορφής αρχείου

Τώρα, θα χρησιμοποιήσουμε το `DetectFileFormat` μέθοδος από το `FileFormatUtil` κλάση για τη συλλογή πληροφοριών σχετικά με τη μορφή αρχείου. Για αυτό το παράδειγμα, υποθέτουμε ότι το κρυπτογραφημένο έγγραφο ονομάζεται "Encrypted.docx" και βρίσκεται στον καθορισμένο κατάλογο:

```csharp
FileFormatInfo info = FileFormatUtil.DetectFileFormat(dataDir + "Encrypted.docx");
```

## Βήμα 4: Ελέγξτε εάν το έγγραφο είναι κρυπτογραφημένο

Για να διαπιστώσουμε εάν το έγγραφο είναι κρυπτογραφημένο, μπορούμε να χρησιμοποιήσουμε το `IsEncrypted` ιδιοκτησία του `FileFormatInfo` αντικείμενο. Αυτή η ιδιότητα επιστρέφει `true` εάν το έγγραφο είναι κρυπτογραφημένο, και `false` διαφορετικά. Θα εμφανίσουμε το αποτέλεσμα στην κονσόλα:

```csharp
Console.WriteLine($"Is the document encrypted? {info.IsEncrypted}");
```
