## Εισαγωγή χώρων ονομάτων

Για να ξεκινήσετε, θα χρειαστεί να εισαγάγετε τους απαραίτητους χώρους ονομάτων από το Aspose.Words. Αυτοί οι χώροι ονομάτων παρέχουν τις κλάσεις και τις μεθόδους που είναι απαραίτητες για τον χειρισμό εγγράφων.

Δείτε πώς μπορείτε να τα εισαγάγετε:

```csharp
using Aspose.Words;
using Aspose.Words.Vba;
```

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Πρώτα απ 'όλα, πρέπει να ορίσουμε τη διαδρομή προς τον κατάλογο του εγγράφου σας. Εδώ θα αποθηκευτεί το έγγραφο προέλευσης του Word και το νέο έγγραφο.

Ορίστε τη διαδρομή προς τον κατάλογό σας ως εξής:

```csharp
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

Φροντίστε να αντικαταστήσετε `"YOUR DOCUMENT DIRECTORY"` με την πραγματική διαδρομή όπου είναι αποθηκευμένα τα έγγραφά σας στο Word. Αυτός ο κατάλογος θα χρησιμεύσει ως χώρος εργασίας μας για αυτό το σεμινάριο.

## Βήμα 2: Φόρτωση του εγγράφου του Word

Αφού ορίσετε τον κατάλογο, ήρθε η ώρα να φορτώσετε το έγγραφο του Word που περιέχει το έργο VBA που θέλετε να κλωνοποιήσετε. Αυτό το βήμα είναι κρίσιμο για την πρόσβαση στο έργο VBA μέσα στο έγγραφο.

Δείτε πώς μπορείτε να φορτώσετε το έγγραφό σας:

```csharp
Document doc = new Document(dataDir + "VBA project.docm");
```

Αυτός ο κώδικας φορτώνει το έγγραφο του Word με το όνομα "VBA project.docm" από τον καθορισμένο κατάλογο στον `doc` αντικείμενο.

## Βήμα 3: Κλωνοποίηση του έργου VBA

Τώρα που έχουμε φορτώσει το αρχικό έγγραφο, ας κλωνοποιήσουμε ολόκληρο το έργο VBA. Αυτό σημαίνει αντιγραφή όλων των ενοτήτων, των αναφορών και των ρυθμίσεων από το αρχικό έγγραφο σε ένα νέο.

Ορίστε ο κώδικας για να το πετύχετε αυτό:

```csharp
Document destDoc = new Document { VbaProject = doc.VbaProject.Clone() };
```

Σε αυτήν τη γραμμή, δημιουργούμε ένα νέο έγγραφο `destDoc` και όρισε το έργο VBA του σε κλώνο του έργου VBA από `doc`Αυτό αντιγράφει όλο το περιεχόμενο VBA από το αρχικό έγγραφο στο νέο.

## Βήμα 4: Αποθήκευση του νέου εγγράφου

Αφού το έργο VBA κλωνοποιηθεί με επιτυχία, το τελευταίο βήμα είναι να αποθηκεύσετε το νέο έγγραφο. Αυτό διασφαλίζει ότι όλες οι αλλαγές σας διατηρούνται και το νέο έγγραφο είναι έτοιμο για χρήση.

Ορίστε ο κώδικας για να αποθηκεύσετε το νέο σας έγγραφο:

```csharp
destDoc.Save(dataDir + "WorkingWithVba.CloneVbaProject.docm");
```

Αυτή η γραμμή αποθηκεύει το νέο έγγραφο με το κλωνοποιημένο έργο VBA ως "WorkingWithVba.CloneVbaProject.docm" στον καθορισμένο κατάλογο.
