## Εισαγωγή

Καλώς ορίσατε στον κόσμο του Aspose.OCR για .NET, ενός εντυπωσιακού εργαλείου που έχει σχεδιαστεί για να ενσωματώνει την Οπτική Αναγνώριση Χαρακτήρων (OCR) στις εφαρμογές σας .NET. Είτε είστε έμπειρος προγραμματιστής είτε ένας περίεργος νεοφερμένος, αυτός ο οδηγός θα σας καθοδηγήσει στα βήματα για να αποκτήσετε ορθογώνια που αναπαριστούν γραμμές από αναγνωρισμένο κείμενο σε εικόνες.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε τα ακόλουθα στη διάθεσή σας:

- Βασικές γνώσεις ανάπτυξης C# και .NET.
- Ένα ολοκληρωμένο περιβάλλον ανάπτυξης (IDE) όπως το Visual Studio.
- Η βιβλιοθήκη Aspose.OCR για .NET είναι εγκατεστημένη. Μπορείτε να την κατεβάσετε. [εδώ](https://releases.aspose.com/ocr/net/).
- Ένα δείγμα εικόνας που περιέχει κείμενο για αναγνώριση.

## Απαιτούμενοι χώροι ονομάτων

Για να ξεκινήσετε, θα χρειαστεί να προσθέσετε τους απαραίτητους χώρους ονομάτων στο έργο σας. Συμπεριλάβετε αυτές τις γραμμές στην αρχή του αρχείου C#:

```csharp
using System;
using System.Collections.Generic;
using System.Drawing;
using System.IO;
using Aspose.OCR;
```

Ακολουθήστε αυτά τα βήματα για να ανακτήσετε ορθογώνια για γραμμές σε μια εικόνα OCR.

## Βήμα 1: Ρύθμιση του καταλόγου εγγράφων σας

Καθορίστε τον κατάλογο όπου βρίσκεται το αρχείο εικόνας σας:

```csharp
// Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "Your Document Directory";
```

Φροντίστε να αντικαταστήσετε `"Your Document Directory"` με την πραγματική διαδρομή.

## Βήμα 2: Αρχικοποίηση του Aspose.OCR

Δημιουργήστε μια παρουσία του `AsposeOcr` κλάση για να αποκτήσετε πρόσβαση στις δυνατότητές της:

```csharp
// Αρχικοποίηση του API Aspose.OCR
AsposeOcr api = new AsposeOcr();
```

## Βήμα 3: Καθορίστε τη διαδρομή εικόνας

Ορίστε την πλήρη διαδρομή προς το αρχείο εικόνας που θέλετε να επεξεργαστείτε:

```csharp
// Καθορίστε την πλήρη διαδρομή προς την εικόνα
string fullPath = dataDir + "sample.png";
```

## Βήμα 4: Αναγνώριση εικόνας και λήψη ορθογωνίων για γραμμές

Τώρα, μπορείτε να χρησιμοποιήσετε το `GetRectangles` μέθοδος για την εξαγωγή ορθογωνίων από αναγνωρισμένες γραμμές κειμένου:

```csharp
// Ανάκτηση ορθογωνίων για γραμμές στην καθορισμένη εικόνα
List<Rectangle> lines = api.GetRectangles(fullPath, AreasType.LINES, false);
```

## Βήμα 5: Εξαγωγή των αποτελεσμάτων

Τέλος, εκτυπώστε τις συντεταγμένες κάθε ανιχνευμένου ορθογωνίου γραμμής στην κονσόλα:

```csharp
// Εμφάνιση των συντεταγμένων των ανιχνευμένων ορθογωνίων
Console.WriteLine("Areas coordinates:");
lines.ForEach(a => Console.WriteLine($"x:{a.X} y:{a.Y} width:{a.Width} height:{a.Height}"));
```

## Σύναψη

Συγχαρητήρια! Ανακτήσατε με επιτυχία ορθογώνια για γραμμές σε μια εικόνα OCR χρησιμοποιώντας το Aspose.OCR για .NET. Αυτή η τεχνολογία ανοίγει πολλές δυνατότητες για εξαγωγή και επεξεργασία κειμένου στις εφαρμογές σας.
