Πριν προχωρήσουμε στην υλοποίηση, βεβαιωθείτε ότι έχετε τα εξής:

2. Δείγμα εικόνας: Προετοιμάστε ένα δείγμα εικόνας (θα χρησιμοποιήσουμε το "sample.png") που θέλετε να αναγνωρίσετε. Βεβαιωθείτε ότι είναι ευκρινές και ευανάγνωστο για βέλτιστα αποτελέσματα OCR.

## Εισαγωγή απαραίτητων χώρων ονομάτων

Ξεκινήστε συμπεριλαμβάνοντας τους απαιτούμενους χώρους ονομάτων στην κορυφή του αρχείου C#:

```csharp
using System;
using System.IO;
using Aspose.OCR;
```

## Βήμα 1: Ρύθμιση καταλόγου εγγράφων

Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας ως εξής:

```csharp
// Ορίστε τη διαδρομή προς τον κατάλογο εγγράφων σας
string dataDir = "Your Document Directory"; // Αντικατάσταση με την πραγματική διαδρομή
```

Βεβαιωθείτε ότι το υποδεικνύετε στην πραγματική τοποθεσία του "sample.png".

## Βήμα 2: Αρχικοποίηση της παρουσίας Aspose.OCR

Δημιουργήστε μια παρουσία του `AsposeOcr` κλάση για πρόσβαση στις λειτουργίες OCR:

```csharp
// Αρχικοποίηση της παρουσίας AsposeOcr
AsposeOcr api = new AsposeOcr();
```

## Βήμα 3: Αναγνώριση εικόνας από ροή

Τώρα, ας αναγνωρίσουμε κείμενο από την εικόνα. Θα ανοίξουμε το αρχείο εικόνας, θα χρησιμοποιήσουμε ένα `MemoryStream`και στη συνέχεια καλέστε τη μέθοδο αναγνώρισης:

```csharp
// Αναγνωρίστε την εικόνα
using (MemoryStream ms = new MemoryStream())
using (FileStream file = new FileStream(Path.Combine(dataDir, "sample.png"), FileMode.Open, FileAccess.Read))
{
    file.CopyTo(ms);
    var result = api.RecognizeImage(ms);
    
    // Εμφάνιση του αναγνωρισμένου κειμένου
    Console.WriteLine("Recognized Text: " + result);
}
```

Αυτό το τμήμα κώδικα διαβάζει την εικόνα σε μια ροή μνήμης και την επεξεργάζεται, επιστρέφοντας το αναγνωρισμένο κείμενο.

## Βήμα 4: Ειδοποίηση επιτυχίας

Επιβεβαιώστε ότι η διαδικασία ολοκληρώθηκε με επιτυχία:

```csharp
Console.WriteLine("Image recognition executed successfully.");
```
