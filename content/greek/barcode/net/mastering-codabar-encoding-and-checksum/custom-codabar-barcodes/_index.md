## Εισαγωγή χώρων ονομάτων

Για να ξεκινήσετε, εισαγάγετε τον απαραίτητο χώρο ονομάτων Aspose στο έργο σας:

```csharp
using Aspose.BarCode.Generation;
```

## Βήμα 1: Αρχικοποίηση της γεννήτριας γραμμωτού κώδικα

Ξεκινήστε δημιουργώντας μια παρουσία του `BarcodeGenerator`, καθορίζοντας τον τύπο γραμμωτού κώδικα ως Codabar και τα δεδομένα που θα κωδικοποιηθούν. Ακολουθεί ένα παράδειγμα:

```csharp
string path = "Your Directory Path"; // Καθορίστε τον κατάλογό σας εδώ
Console.WriteLine("Generating Codabar with Start/Stop Characters:");

BarcodeGenerator gen = new BarcodeGenerator(EncodeTypes.Codabar, "-12345-");
```

Αντικαθιστώ `"-12345-"` με τα δεδομένα που θέλετε να κωδικοποιήσετε.

## Βήμα 2: Ορίστε τη Διάσταση Χ

Η διάσταση Χ ορίζει το πλάτος των στοιχείων του γραμμωτού κώδικα, μετρούμενο σε pixel. Προσαρμόστε το σύμφωνα με τις απαιτήσεις σας:

```csharp
gen.Parameters.Barcode.XDimension.Pixels = 2; // Αλλαγή όπως απαιτείται
```

## Βήμα 3: Ορισμός χαρακτήρων έναρξης και λήξης

Το Codabar υποστηρίζει διάφορους χαρακτήρες έναρξης και λήξης - A, B, C και D. Ορίστε αυτά τα σύμβολα με βάση τις συγκεκριμένες απαιτήσεις σας. Παρακάτω παρατίθενται παραδείγματα για κάθε χαρακτήρα:

### Έναρξη Α και διακοπή Α:

```csharp
gen.Parameters.Barcode.Codabar.CodabarStartSymbol = CodabarSymbol.A;
gen.Parameters.Barcode.Codabar.CodabarStopSymbol = CodabarSymbol.A;
```

### Έναρξη Β και Διακοπή Β:

```csharp
gen.Parameters.Barcode.Codabar.CodabarStartSymbol = CodabarSymbol.B;
gen.Parameters.Barcode.Codabar.CodabarStopSymbol = CodabarSymbol.B;
```

### Έναρξη C και διακοπή C:

```csharp
gen.Parameters.Barcode.Codabar.CodabarStartSymbol = CodabarSymbol.C;
gen.Parameters.Barcode.Codabar.CodabarStopSymbol = CodabarSymbol.C;
```

### Έναρξη D και διακοπή D:

```csharp
gen.Parameters.Barcode.Codabar.CodabarStartSymbol = CodabarSymbol.D;
gen.Parameters.Barcode.Codabar.CodabarStopSymbol = CodabarSymbol.D;
```

Επιλέξτε τα κατάλληλα σύμβολα με βάση τις ανάγκες της εφαρμογής σας.

## Βήμα 4: Αποθήκευση των δημιουργημένων εικόνων γραμμωτού κώδικα

Τέλος, αποθηκεύστε τις δημιουργημένες εικόνες γραμμωτού κώδικα Codabar στον καθορισμένο κατάλογο:

```csharp
gen.Save($"{path}CodabarStartAStopA.png", BarCodeImageFormat.Png);
gen.Save($"{path}CodabarStartBStopB.png", BarCodeImageFormat.Png);
gen.Save($"{path}CodabarStartCStopC.png", BarCodeImageFormat.Png);
gen.Save($"{path}CodabarStartDStopD.png", BarCodeImageFormat.Png);
```

Αυτό θα δημιουργήσει τέσσερις διαφορετικές εικόνες γραμμωτού κώδικα με τους καθορισμένους χαρακτήρες έναρξης και λήξης.
